RESUME SYNOPTIQUE DU PROTOCOLE

+----------------------------------+----------------------------------+
| 1.  Titre complet                | Modélisation de la circulation   |
|                                  | du mpox, identification de       |
|                                  | groupes à risque d’infection, et |
|                                  | analyse coût-efficacité de       |
|                                  | stratégies vaccinales en         |
|                                  | République Centrafricaine        |
+----------------------------------+----------------------------------+
| 2.  Acronyme                     | VAXPOX-RCA                       |
+----------------------------------+----------------------------------+
| 3.  Promoteur                    | Institut Pasteur de Bangui (IPB) |
+----------------------------------+----------------------------------+
| 4.  Responsables scientifiques   | Nicolas BERTHET, IPB             |
|                                  |                                  |
|                                  | Arnaud FONTANET, IPP             |
+----------------------------------+----------------------------------+
| 5.  Type d’étude                 | Etude sur échantillons           |
|                                  | biologiques d’origine humaine    |
|                                  | avec données personnelles de     |
|                                  | santé, basée sur une approche    |
|                                  | pluridisciplinaire comprenant 4  |
|                                  | volets : épidémiologie,          |
|                                  | sérologie, virologie et          |
|                                  | modélisation mathématique.       |
+----------------------------------+----------------------------------+
| 6.  Rationnel de l’étude         | En 2022 puis en 2024,            |
|                                  | l’Organisation Mondiale de la    |
|                                  | Santé (OMS) a déclaré le mpox    |
|                                  | “Urgence de Santé Publique de    |
|                                  | Portée Internationale” en raison |
|                                  | de l’émergence de sous-clades du |
|                                  | virus de la variole du singe     |
|                                  | (MPXV), de leur propagation vers |
|                                  | des régions et des pays          |
|                                  | jusque-là non touchés, y compris |
|                                  | dans des zones urbaines, et la   |
|                                  | transmission interhumaine        |
|                                  | soutenue, y compris par voie     |
|                                  | sexuelle. Des millions de doses  |
|                                  | de vaccins (formes atténuées du  |
|                                  | vaccin contre la variole         |
|                                  | conférant une protection croisée |
|                                  | contre le MPXV) vont être        |
|                                  | distribuées aux pays africains   |
|                                  | les plus touchés pour contrer    |
|                                  | l’épidémie. Il est urgent de     |
|                                  | disposer de données              |
|                                  | scientifiques pour mieux         |
|                                  | orienter les stratégies          |
|                                  | vaccinales et identifier les     |
|                                  | populations à prioriser pour la  |
|                                  | vaccination.                     |
|                                  |                                  |
|                                  | Le projet VAXPOX a pour objectif |
|                                  | d’accompagner les politiques de  |
|                                  | santé publique à mesure que les  |
|                                  | vaccins deviennent disponibles,  |
|                                  | via une approche                 |
|                                  | pluridisciplinaire associant     |
|                                  | épidémiologie, sérologie,        |
|                                  | virologie moléculaire et         |
|                                  | modélisation mathématique. Ce    |
|                                  | projet se décline dans 6 pays    |
|                                  | hébergeant un institut du        |
|                                  | Pasteur Network (PN), avec des   |
|                                  | activités adaptées selon les     |
|                                  | situations épidémiologiques au   |
|                                  | sein de chaque pays.             |
|                                  |                                  |
|                                  | Le protocole VAXPOX-RCA a pour   |
|                                  | objectif de présenter les        |
|                                  | activités de recherche qui       |
|                                  | seront conduites en République   |
|                                  | Centrafricaine (RCA) par         |
|                                  | l’Institut Pasteur de Bangui     |
|                                  | (IPB), laboratoire national de   |
|                                  | référence (LNR) pour le          |
|                                  | diagnostic mpox. L’ensemble des  |
|                                  | WP seront déclinés dans ce pays  |
|                                  | qui connaît des épidémies de     |
|                                  | mpox depuis 40 ans, et a         |
|                                  | rapporté pour la première fois   |
|                                  | fin 2024 des cas en zone         |
|                                  | urbaine, à Bangui sa capitale.   |
+----------------------------------+----------------------------------+
| 7.  Objectif principal           | -   Estimer la séroprévalence    |
|                                  |       des anticorps anti-MPXV    |
|                                  |       par âge, sexe et groupe à  |
|                                  |       risque dans différentes    |
|                                  |       populations urbaines et    |
|                                  |       rurales en RCA             |
+----------------------------------+----------------------------------+
| 8.  Objectifs secondaires        | -   Identifier les facteurs      |
|                                  |       associés à l’infection par |
|                                  |       le MPXV selon les          |
|                                  |       caractéristiques des       |
|                                  |       sujets participant à       |
|                                  |       l’étude                    |
|                                  |                                  |
|                                  | -   Estimer la prévalence de     |
|                                  |       l’intention vaccinale dans |
|                                  |       différentes populations    |
|                                  |       urbaines et rurales en RCA |
|                                  |                                  |
|                                  | -   Estimer la séroprévalence    |
|                                  |       des anticorps dirigés      |
|                                  |       contre un panel de         |
|                                  |       maladies infectieuses      |
|                                  |       d’intérêt en santé         |
|                                  |       publique dans différentes  |
|                                  |       populations urbaines et    |
|                                  |       rurales en RCA             |
|                                  |                                  |
|                                  | -   Estimer le taux de           |
|                                  |       transmission du MPXV au    |
|                                  |       sein de ménages en RCA     |
|                                  |                                  |
|                                  | -   Quantifier les contributions |
|                                  |       respectives de la          |
|                                  |       transmission zoonotique et |
|                                  |       de la transmission         |
|                                  |       interhumaine du MPXV dans  |
|                                  |       la dissémination du mpox   |
|                                  |       en RCA                     |
|                                  |                                  |
|                                  | -   Modéliser la circulation du  |
|                                  |       mpox en RCA sur la base    |
|                                  |       des données collectées     |
|                                  |                                  |
|                                  | -   Identifier les stratégies    |
|                                  |       vaccinales les plus        |
|                                  |       efficientes dans le        |
|                                  |       contexte de la RCA         |
+----------------------------------+----------------------------------+
| 9.  Critère d’évaluation         | -   Séroprévalence des anticorps |
|     principal                    |       IgM et IgG anti-MPXV par   |
|                                  |       âge, sexe et groupe à      |
|                                  |       risque dans différentes    |
|                                  |       populations urbaines et    |
|                                  |       rurales en RCA             |
+----------------------------------+----------------------------------+
| 10. Critères d’évaluation        | -   Facteur d’augmentation (ou   |
|     secondaires                  |       de diminution) du risque   |
|                                  |       d’infection passée par le  |
|                                  |       MPXV selon les             |
|                                  |       caractéristiques des       |
|                                  |       sujets participant à       |
|                                  |       l’étude                    |
|                                  |                                  |
|                                  | -   Prévalence de l’intention    |
|                                  |       vaccinale dans différentes |
|                                  |       populations urbaines et    |
|                                  |       rurales en RCA             |
|                                  |                                  |
|                                  | -   Séroprévalence des anticorps |
|                                  |       dirigés contre un panel de |
|                                  |       maladies infectieuses      |
|                                  |       d’intérêt en santé         |
|                                  |       publique dans différentes  |
|                                  |       populations urbaines et    |
|                                  |       rurales en RCA             |
|                                  |                                  |
|                                  | -   Taux de transmission du MPXV |
|                                  |       au sein de ménages de cas  |
|                                  |       confirmés de mpox en RCA   |
|                                  |                                  |
|                                  | -   Part relative des            |
|                                  |       introductions zoonotiques  |
|                                  |       versus cas de transmission |
|                                  |       interhumaine en RCA        |
|                                  |                                  |
|                                  | -   Modèle mathématique          |
|                                  |       décrivant la circulation   |
|                                  |       du mpox en RCA, calibré    |
|                                  |       aux données collectées     |
|                                  |       dans le projet             |
|                                  |                                  |
|                                  | -   Résultats d’analyses         |
|                                  |       coût-efficacité de         |
|                                  |       différentes stratégies     |
|                                  |       vaccinales en RCA          |
+----------------------------------+----------------------------------+
| 11. Echantillons biologiques     | -   Un prélèvement de sang       |
|                                  |       réalisé chez les           |
|                                  |       participants au cours des  |
|                                  |       enquêtes de séroprévalence |
|                                  |       (groupes représentatifs de |
|                                  |       la population générale,    |
|                                  |       groupes à risques de       |
|                                  |       transmission par voie      |
|                                  |       sexuelle) : 2.5 ml de sang |
|                                  |       veineux chez les           |
|                                  |       participants de 6 ans ou   |
|                                  |       plus, ou jusqu’à 500 µl de |
|                                  |       sang capillaire chez les   |
|                                  |       participants de 1 à 5 ans. |
|                                  |                                  |
|                                  | -   Ecouvillons de pus, croûte,  |
|                                  |       sang ou prélèvements de    |
|                                  |       gorge issus de la          |
|                                  |       surveillance des cas       |
|                                  |       suspects de mpox (cas      |
|                                  |       passés et cas qui seraient |
|                                  |       détectés pendant la durée  |
|                                  |       de l’étude et membres de   |
|                                  |       ménages de cas confirmés). |
+----------------------------------+----------------------------------+
| 12. Méthode - déroulement de la  | Le projet VAXPOX-RCA s’articule  |
|     recherche                    | autour de 4 Work Packages (WP) : |
|                                  |                                  |
|                                  | WP1 Epidémiologie et WP2         |
|                                  | Sérologie : réalisation          |
|                                  | d’enquêtes transversales de      |
|                                  | séroprévalence auprès de         |
|                                  | populations représentatives de   |
|                                  | la population générale et auprès |
|                                  | de sujets appartenant à des      |
|                                  | groupes particulièrement à       |
|                                  | risque de transmission par voie  |
|                                  | sexuelle. Ces enquêtes           |
|                                  | comprennent un questionnaire     |
|                                  | standardisé et un prélèvement    |
|                                  | sanguin pour chaque participant. |
|                                  | La détection des anticorps       |
|                                  | spécifiques anti-MPXV dans les   |
|                                  | échantillons de sérum témoignera |
|                                  | d’une infection passée et        |
|                                  | permettra de déterminer les      |
|                                  | populations les plus affectées   |
|                                  | par le mpox. Aussi, l’analyse    |
|                                  | des échantillons de sang         |
|                                  | prélevés dans le cadre de la     |
|                                  | surveillance auprès des membres  |
|                                  | des ménages des cas confirmés de |
|                                  | mpox permettra de déterminer le  |
|                                  | taux de transmission du virus au |
|                                  | sein des ménages.                |
|                                  |                                  |
|                                  | WP3 Virologie moléculaire :      |
|                                  | séquençage des souches virales   |
|                                  | isolées des cas confirmés de     |
|                                  | mpox, à partir des échantillons  |
|                                  | issus de la surveillance         |
|                                  | nationale en RCA, pour           |
|                                  | déterminer le caractère          |
|                                  | zoonotique ou interhumain de la  |
|                                  | transmission du MPXV.            |
|                                  |                                  |
|                                  | WP4 Modélisation mathématique :  |
|                                  | élaboration de modèles           |
|                                  | mathématiques décrivant la       |
|                                  | circulation du mpox en RCA,      |
|                                  | calibrés grâce aux données       |
|                                  | collectées dans les autres WP ;  |
|                                  | réalisation d’analyses           |
|                                  | coût-efficacité de stratégies    |
|                                  | vaccinales pour identifier les   |
|                                  | plus efficientes dans le         |
|                                  | contexte de la RCA.              |
+----------------------------------+----------------------------------+
| 13. Populations étudiées         | -   Sujets représentatifs de la  |
|                                  |       population générale en     |
|                                  |       zone rurale (patients de   |

Justification scientifique DE LA RECHERCHE

Contexte. La variole du singe, à présent appelée mpox, a été rapportée
chez l'homme pour la première fois en 1970. Jusqu’en 2022, le virus
causant cette maladie (MPXV) ne provoquait que des épidémies limitées
d’origine zoonotique après contact avec des animaux sauvages, notamment
des écureuils (Funisciurus) et des rats (Cricetomys), et une
transmission interhumaine restreinte, principalement par des contacts
étroits avec les fluides biologiques et les lésions cutanées des
personnes infectées¹. Deux clades génétiques du MPXV ont été
initialement décrits : le clade I en Afrique centrale et le clade II en
Afrique de l’Ouest². Jusqu’en 2017, le clade I prédominait et était
connu pour causer une maladie plus sévère que le clade II, avec un taux
de létalité (CFR) estimé à 10,6 %. Le clade II était considéré comme
moins sévère, avec un CFR de 3,6 %³. Le CFR variait selon l’âge : il
était estimé à 9,7 % chez les enfants de 5 ans et à 1,2 % chez les
sujets de 30 ans pour le clade I⁴. Après quatre décennies sans nouveaux
cas rapportés, le MPXV clade II a réémergé au Nigeria en 2017⁵, suivi
d’une pandémie en 2022 touchant des pays non endémiques, avec des
transmissions interhumaines soutenues d’un sous-clade appelé IIb,
principalement par contacts sexuels entre hommes ayant des relations
sexuelles avec des hommes (HSH^(6,7)). En juillet 2022, l’Organisation
mondiale de la santé (OMS) a déclaré le mpox comme une Urgence de Santé
Publique de Portée Internationale (USPPI). Cette urgence a duré un an,
avec près de 100 000 cas rapportés dans plus de 100 pays et un CFR
inférieur à 0,1 %⁸. Des campagnes de prévention et de vaccination
ciblées auprès des populations à haut risque de transmission par voie
sexuelle, principalement les HSH, ont permis de contenir
l’épidémie^(9,10). Alors que cette épidémie due au clade IIb déclinait,
les cas dus au clade I ont progressivement augmenté en République
Démocratique du Congo (RDC) et le virus s’est propagé à des pays
d’Afrique de l’Est jusque-là épargnés, y compris dans des zones
urbaines. Depuis 2024, la dynamique de l’épidémie en Afrique suggère
pour les pays nouvellement touchés l’introduction du virus via les
populations à haut risque de transmission par voie sexuelle, notamment
dans la classe d’âge 15-44 ans, les enfants pouvant être infectés
secondairement^(11,12). D‘août 2024 à septembre 2025, l’OMS a de nouveau
déclaré le mpox comme USPPI¹³, en raison de l'apparition d’un nouveau
sous-clade dénommé clade Ib, de la transmission interhumaine soutenue
observée (y compris par voie sexuelle¹⁴) et de la diffusion du MPXV de
régions historiquement rurales et forestières vers de nouvelles zones, y
compris urbaines¹⁵. Depuis, plusieurs pays africains ont rapporté une
transmission communautaire soutenue du clade Ib, et 14 pays en-dehors du
continent africain ont rapporté des cas sporadiques liés à des retours
de voyage^(16,17). La levée de l’alerte USPPI en septembre 2025 ne
marque pas la fin de la lutte contre le mpox, avec de nombreuses
épidémies toujours en cours sur le continent africain.

Vaccins disponibles. La réponse à cette épidémie doit être adaptée aux
caractéristiques épidémiologiques et cliniques spécifiques des
différentes épidémies simultanées sur le continent africain¹⁸. Dans ce
contexte, il est important d’examiner le rôle des vaccins parmi les
outils de prévention¹⁹, particulièrement après les résultats décevants
d’un essai contrôlé randomisé de phase III portant sur le seul
médicament disponible, le Tecovirimat, en RDC²⁰. Le vaccin utilisé lors
de la pandémie de 2022, Modified Vaccinia Ankara (MVA), est une forme
atténuée du vaccin antivariolique fabriqué par Bavarian Nordic. Son
efficacité contre le MPXV clade IIb a été estimée à 76 % après une dose
et 82 % après deux doses²¹. Bien que son efficacité contre d’autres
clades du MPXV ne soit pas encore connue, une protection croisée est
attendue, comme cela a été observé avec les vaccins antivarioliques de
première génération, qui conféraient une protection de 85 % contre le
mpox²². Le vaccin MVA a reçu une préqualification de l’OMS pour une
utilisation uniquement chez les enfants de plus de 12 ans. Un autre
vaccin, appelé LC16m8, développé il y a plus de 20 ans au Japon, a reçu
le 19 novembre 2024 une autorisation d’utilisation d’urgence de l’OMS
pour les adultes et les enfants de plus d’un an, en dose unique²³, avec
plus de 3 millions de doses offertes par le gouvernement japonais. En
1973-1974, ce vaccin a été administré à environ 50 000 enfants japonais
âgés de 1 à 7 ans, avec quelques effets secondaires cutanés et des
épisodes d’auto-inoculation²⁴. Des questions de responsabilité juridique
pourraient retarder la livraison des doses aux pays concernés, et son
administration avec des aiguilles bifurquées pourrait réduire son
acceptabilité. En janvier 2025, l’Autorité de Régulation Pharmaceutique
de la RDC a autorisé l’inclusion de tous les enfants à partir d’1 an
dans les campagnes de vaccination. Enfin, en date de février 2025, six
millions de doses ont été promises au continent africain¹⁷.

Stratégies vaccinales. Bien que ces vaccins élargissent l’arsenal pour
freiner la propagation du mpox, les stratégies à mettre en place restent
incertaines. Étant donné les stocks limités de vaccins, une vaccination
généralisée n'est pas envisageable. La stratégie recommandée par le CDC
Africa, l’OMS et d’autres organisations est de cibler les contacts de
cas confirmés, les professionnels de santé, les autres acteurs de
première ligne, et les populations clé à risque d’infection dans des
zones avec transmission interhumaine active²⁵. Il est urgent de disposer
de données scientifiques afin d’éclairer au mieux ces stratégies et
d’identifier les populations clés à cibler par les campagnes de
vaccination. Les connaissances actuelles sur la transmission du MPXV
identifient les groupes potentiels suivants à cibler pour vaccination :
les professionnels de santé exposés aux patients ; en zone rurale, les
enfants en contact avec les réservoirs animaux (souvent via le piégeage
d’animaux sauvages ou d’autres activités) qui sont plus à risque de
formes sévères de la maladie ; en zone urbaine, les populations à risque
élevé de transmission par voie sexuelle (travailleurs du sexe, HSH,
patients atteints d’infections sexuellement transmissibles (IST),
chauffeurs routiers). Cependant, il n’est pas précisément établi quel(s)
groupe(s) à risque devraient être priorisés pour les campagnes de
vaccination pour un impact maximal dans un contexte de limitation de la
quantité de vaccins. Une étude a recommandé la vaccination des enfants
dans les régions endémiques de la RDC²⁶, mais la migration urbaine
récente du virus et sa transmission soutenue dans les réseaux sexuels²⁷
n’ont pas été pris en compte. À la suite de cette publication, la
nécessité d’analyses coût-efficacité de différentes stratégies
vaccinales a été mise en avant par plusieurs experts^(28–31). Néanmoins,
cette évaluation essentielle pour orienter les décisions de santé
publique semble impossible en l’absence de données sur le fardeau réel
de l’infection dans les groupes à risque, et les contributions
respectives des différentes voies de transmission. Aussi, la littérature
indique que l’acceptabilité vaccinale mpox en Afrique est parmi les plus
faibles au monde^(32,33), mais des données nationales ne sont pas
disponibles pour tous les pays. Mesurer l’intention de vaccination parmi
les groupes à risque d’infection est essentiel pour favoriser une
couverture vaccinale optimale.

Situation épidémiologique en République Centrafricaine. En République
Centrafricaine (RCA), le Laboratoire National de Référence (LNR) pour le
mpox est hébergé par l’Institut Pasteur de Bangui (IPB). Depuis des
épidémies mineures rapportées en 1984 et 2001^(34,35), 264 cas de mpox
ont été confirmés en laboratoire, principalement en provenance de zones
rurales frontalières de la RDC. En 2024, 92 cas confirmés ont été
répertoriés, le nombre de cas le plus élevé jamais rapporté en une année
en RCA. Parmi eux, 9 cas ont été détectés à Bangui, la capitale. En
2025, 73 cas confirmés ont déjà été enregistrés. Les données de
séquençage de 2001 à 2024 montrent que de nombreux cas confirmés sont
phylogénétiquement proches des souches de clade Ia isolées en RDC et
dans la forêt tropicale du bassin du Congo^(36,37). On ne peut pas
écarter cependant que la circulation du virus en RCA soit plus
importante encore, soit du fait du caractère reculé des zones rurales où
les transmissions d’origine zoonotique ont lieu, soit du fait de formes
cliniques pauci-symptomatiques, notamment lors des transmissions par
voie sexuelle en milieu urbain. Des enquêtes sérologiques permettant
l’identification d’infections passées et parfois peu symptomatiques
seraient un complément important de la surveillance de la circulation du
MPXV en RCA.

Objectif du projet. Le projet VAXPOX a pour objectif d’accompagner les
politiques de santé publique à mesure que les vaccins deviennent
disponibles, via une approche pluridisciplinaire associant épidémiologie
(WP1), sérologie (WP2), virologie moléculaire (WP3) et modélisation
mathématique comprenant une analyse coût-efficacité (WP4). Ce projet se
décline dans 6 pays hébergeant un institut du Pasteur Network (PN), avec
des activités adaptées selon les situations épidémiologiques au sein de
chaque pays. Le protocole VAXPOX-RCA a pour objectif de présenter les
activités de recherche qui seront conduites en RCA par l’IPB. L’ensemble
des WP seront déclinés en RCA, qui connaît des épidémies de mpox depuis
40 ans avec une forte augmentation du nombre de cas confirmés ces deux
dernières années, et a récemment rapporté pour la première fois des cas
en zone urbaine, à Bangui sa capitale.

En parallèle de ces objectifs spécifiques au mpox, le projet VAXPOX-RCA
sera l’occasion d’évaluer la circulation d’autres pathogènes d’intérêt
dans les différents pays déclinant ce projet. En effet, de nombreux
pathogènes d’intérêt pour la santé des populations circulent souvent
dans les mêmes zones des pays à revenu faible ou intermédiaire. Les
nouveaux outils de recherche que sont les tests sérologiques multiplex
permettent une description détaillée de la cinétique des anticorps
spécifiques de plusieurs pathogènes. Le projet VAXPOX-RCA aura aussi
pour objectif d’évaluer la circulation de plusieurs agents infectieux
d’intérêt parmi les populations ciblées dans cette étude.

IMPACTS ATTENDUS DU PROJET

Impacts pour la communauté scientifique

Les résultats des analyses effectuées au sein de chacun des WP du projet
VAXPOX-RCA fourniront des données pertinentes pour une meilleure
compréhension de l’épidémiologie du mpox. L’obtention de données de
séroprévalence en population générale et dans des groupes
particulièrement à risque de transmission du virus par voie sexuelle
(WP1-WP2) permettra de quantifier la circulation d’un virus
historiquement restreint à des zones reculées forestières mais ayant à
présent atteint de nouvelles zones géographiques, y compris des zones
urbaines et péri-urbaines qui maximisent la transmission interhumaine.
Grâce à la technologie multiplex, le projet VAXPOX-RCA permettra
également d’évaluer la circulation de plusieurs agents infectieux
d’intérêt pour la santé des populations, au sein des groupes ciblés par
cette étude.

Les analyses phylogénétiques des souches virales circulant en RCA et
dans les pays voisins (WP3) permettront une meilleure compréhension de
l’évolution du MPXV, ainsi que la détection des nouveaux sous-clades
circulant actuellement sur le continent africain et de sa dissémination
dans la région. Enfin, le modèle mathématique développé pour décrire la
circulation du mpox en RCA, prenant notamment en compte les différents
modes de transmission (WP4) pourra être adapté à d’autres contextes pour
mieux comprendre la dissémination de ce virus.

Impacts communautaires

Le contrôle de l’épidémie actuelle de mpox est d’une importance capitale
pour les populations, en particulier pour les enfants qui sont les plus
à risque de formes graves de la maladie, mais aussi pour prévenir
l’établissement du mpox en zone urbaine via une transmission
interhumaine soutenue. La réalisation du projet VAXPOX-RCA contribuera à
mieux identifier les groupes particulièrement à risque d’infection par
le MPXV, et permettra de cibler les campagnes de vaccination afin
qu’elles soient les plus efficientes pour limiter la dissémination du
virus. De plus, la communication des résultats du projet (cf. Section 9.
Diffusion des résultats) permettra de sensibiliser les populations à la
contribution relative de la transmission zoonotique et la transmission
interhumaine du MPXV, ainsi qu’au recours à la vaccination pour se
protéger de la maladie. Aussi, l’utilisation de la technologie multiplex
pour évaluer la circulation de plusieurs agents pathogènes présente un
avantage substantiel pour les populations ciblées, car la fourniture
d'un seul échantillon sanguin dans le cadre du projet VAXPOX-RCA
contribuera à la surveillance d'un éventail de maladies infectieuses,
sans nécessiter la mise en place de multiples études.

Impacts en santé publique

Les résultats du projet VAXPOX-RCA seront particulièrement pertinents
pour les autorités sanitaires de RCA dans le contexte de l’épidémie en
cours, en documentant la progression de l’épidémie de mpox au sein des
populations ciblées par cette étude, et en identifiant des groupes
particulièrement à risque d’infection. Grâce à la modélisation et à
l’analyse coût-efficacité des stratégies vaccinales, il sera possible de
conseiller les autorités sanitaires sur les groupes à vacciner en
priorité afin de maximiser l’impact des campagnes. Le projet VAXPOX-RCA
aura également un impact à long terme car les recommandations pourront
être adaptées à de nouvelles informations (disponibilité des vaccins,
acceptabilité vaccinale, données actualisées de séroprévalence…) en
mettant à jour les modèles mathématiques développés. Enfin, les
résultats de sérologie multiplex contre un panel d’agents infectieux
d’intérêt en santé publique éclaireront les autorités sanitaires sur la
prévalence de l'exposition à plusieurs maladies infectieuses courantes
parmi les populations ciblées par cette étude.

Impacts sur la préparation aux pandémies

Le projet VAXPOX-RCA permettra de renforcer les compétences de l’IPB
concernant des méthodes et des technologies cruciales pour la réponse
aux menaces infectieuses à potentiel pandémique. Premièrement, la
réalisation d’enquêtes sérologiques de terrain avec une méthodologie
hautement standardisée et adaptable à d’autres pathogènes émergents
(ex : technologie Luminex) est essentielle à la compréhension de la
diffusion d’un pathogène au sein de groupes à risque et en population
générale. Ces enquêtes peuvent avoir plusieurs objectifs :
identification des populations à risque, mise en évidence de formes
cliniques asymptomatiques, évaluation de l’immunité acquise après
infection/vaccination, et étude des chaînes de transmission autour des
cas confirmés. Deuxièmement, le séquençage systématique du génome
complet, le partage des séquences d’agents pathogènes au niveau régional
et les analyses phylogénétiques groupées de ces séquences permettent
d’assurer une surveillance quasi en temps réel de la circulation d’un
agent pathogène donné et de sa capacité d’échappement immunitaire,
améliorant ainsi la coordination régionale pour la surveillance grâce à
une couverture et un reporting accrus. Troisièmement, le renforcement
des compétences en modélisation mathématique des épidémies est un
complément essentiel à une meilleure compréhension des risques
épidémiques et des réponses à apporter aux menaces infectieuses. Enfin,
la collaboration entre chercheurs de différents instituts du PN sera
source de bénéfices multilatéraux. Les collaborations nouées dans le
cadre de ce projet de recherche seront déterminantes pour faciliter les
interactions en période de pandémie due à un autre pathogène, permettant
un partage d’expertise au bénéfice de l’ensemble du réseau.

Objectifs

Objectif principal

L’objectif principal de ce projet est d’estimer la séroprévalence des
anticorps anti-MPXV par âge, sexe et groupe à risque dans différentes
populations urbaines et rurales en RCA.

Objectifs secondaires

Les objectifs secondaires de ce projet sont définis ci-dessous :

-   Identifier les facteurs associés à l’infection par le MPXV dans
      différentes populations urbaines et rurales en RCA

-   Estimer la prévalence de l’intention vaccinale mpox dans différentes
      populations urbaines et rurales en RCA

-   Estimer la séroprévalence des anticorps dirigés contre un panel de
      maladies infectieuses d’intérêt en santé publique dans différentes
      populations urbaines et rurales en RCA

-   Estimer le taux de transmission du MPXV au sein de ménages en RCA

-   Quantifier les contributions respectives de la transmission
      zoonotique et de la transmission interhumaine du MPXV dans la
      dissémination du mpox en RCA

-   Modéliser la circulation du mpox en RCA sur la base des données
      collectées

-   Identifier les stratégies vaccinales les plus efficientes dans le
      contexte de la RCA

METHODOLOGIE

1.  

Critères d’évaluation

Critère d’évaluation principal

Séroprévalence des anticorps IgM et IgG anti-MPXV par âge, sexe et
groupe à risque dans différentes populations urbaines et rurales en RCA.

Critères d’évaluation secondaires

-   Facteur d’augmentation (ou de diminution) du risque d’infection
      passée par le MPXV selon les caractéristiques des sujets
      participant à l’étude

-   Prévalence de l’intention vaccinale mpox dans différentes
      populations urbaines et rurales en RCA

-   Séroprévalence des anticorps dirigés contre un panel de maladies
      infectieuses d’intérêt en santé publique dans différentes
      populations urbaines et rurales en RCA

-   Taux de transmission du MPXV au sein de ménages de cas confirmés de
      mpox en RCA

-   Part relative des introductions zoonotiques versus cas de
      transmission interhumaine en RCA

-   Modèle mathématique décrivant la circulation du mpox en RCA, calibré
      aux données collectées dans le projet

-   Résultats d’analyses coût-efficacité de différentes stratégies
      vaccinales en RCA

Plan expérimental

WP1-2 : Epidémiologie et sérologie

Objectif principal

Pour estimer la séroprévalence des anticorps anti-MPXV par âge, sexe et
groupe à risque dans différentes populations urbaines et rurales en RCA,
des enquêtes de séroprévalence seront menées auprès des populations
suivantes :

-   Sujets représentatifs de la population générale adulte en zone
    urbaine (professionnels de santé du centre hospitalier et
    universitaire communautaire de Bangui (CHUCB)),

-   Sujets représentatifs de la population générale en zone rurale
    (patients de l’hôpital de Mbaiki, zone rapportant régulièrement des
    cas de mpox confirmés en laboratoire),

-   Sujets issus de groupes particulièrement à risque de transmission du
    MPXV par voie sexuelle à Bangui (travailleurs du sexe, et patients
    du Centre National de Référence des Infections Sexuellement
    Transmissibles et de la Thérapie Antirétrovirale (CNRISTAR) de
    Bangui),

Ces enquêtes impliqueront :

-   Un questionnaire standardisé pour recueillir des données
    socio-démographiques, épidémiologiques et cliniques,

-   Un prélèvement de sang (2.5 ml de sang veineux pour les participants
    de 6 ans ou plus, ou jusqu’à 500 µl de sang capillaire pour les
    participants de 1 à 5 ans).

Les analyses sérologiques des échantillons collectés détecteront les
éventuelles expositions passées au MPXV grâce à un test immunologique
multiplex sur microsphères (MMIA) basé sur la technologie Luminex,
permettant la détection simultanée des anticorps IgM ou IgG dirigés
contre plus de 35 antigènes du MPXV. Ce test a été développé à l’IPP et
sera transféré aux équipes de l’IPB, qui disposent de la technologie et
l’appareillage nécessaires pour procéder aux analyses. Les données
sérologiques mpox obtenues et les données épidémiologiques associées
seront partagées dans une base de données commune avec les autres
instituts du PN déclinant le projet VAXPOX, permettant d’élargir la
surveillance collaborative entre les instituts du réseau à un volet
sérologique.

Objectifs secondaires

Afin de répondre à plusieurs objectifs secondaires de cette recherche,
les éléments suivants seront mis en œuvre :

-   Identifier les facteurs associés à l’infection par le MPXV selon les
    groupes à risque en RCA : Dans l’éventualité où des sous-groupes
    parmi les populations ciblées seraient séropositifs pour le mpox,
    les données collectées dans le questionnaire administré aux
    participants permettront d’évaluer les facteurs d’augmentation (ou
    de diminution) du risque d’infection passée par le MPXV selon les
    caractéristiques des participants.

-   Estimer la prévalence de l’intention vaccinale mpox dans différentes
    populations urbaines et rurales en RCA : Le questionnaire administré
    aux participants comportera des questions se rapportant à
    l’acceptabilité anticipée du vaccin mpox, ce qui permettra d’évaluer
    la prévalence de l’intention vaccinale dans les populations ciblées.

-   Estimer la séroprévalence des anticorps dirigés contre un panel de
    maladies infectieuses d’intérêt en santé publique : La technologie
    multiplex Luminex utilisée pour répondre à l’objectif principal
    repose sur l’utilisation de billes enrobées d’antigènes, permettant
    de mesurer avec précision un grand nombre d’anticorps à partir d’un
    petit volume de sérum ou de plasma. Le test utilisé pour le projet
    permettra d’évaluer la prévalence des anticorps spécifiques contre
    un panel d’agents pathogènes d’intérêt, dans les populations
    ciblées.

-   Estimer le taux de transmission du MPXV au sein de ménages en RCA :
    Dans le cadre des activités de surveillance nationale du mpox
    auxquelles participe l’IPB, des investigations d’épidémie sont
    habituellement conduites au sein des ménages des cas confirmés, au
    cours desquelles des prélèvements de sang sont effectués. Les
    échantillons issus de ces enquêtes seront analysés pour détecter
    l’éventuelle présence d’anticorps spécifiques anti-MPXV, ce qui
    permettra d’identifier tous les éventuels cas secondaires au sein du
    ménage. 

WP3 : Virologie moléculaire

Pour répondre à l’objectif secondaire « quantifier les contributions
respectives de la transmission zoonotique et de la transmission
interhumaine du MPXV » dans la dissémination du mpox en RCA, des
analyses génomiques seront menées à partir d’échantillons (pus, croûtes
de lésions cutanées, sang et prélèvements de gorge) et leurs données
associées, issus de la surveillance nationale à laquelle l’IPB prend
part en tant que LNR pour le diagnostic mpox. Les analyses conduites
dans le cadre de la recherche incluront le séquençage complet des
souches de MPXV de cas confirmés de mpox et des analyses
phylogénétiques.

1.  Les échantillons provenant de cas confirmés détectés par le système
      de surveillance pendant la durée de la recherche et présentant un
      seuil de cycle PCR MPXV inférieur ou égal à 30 seront sélectionnés
      en priorité pour séquençage à l’IPB afin d’obtenir des génomes
      complets. Les échantillons issus de cas confirmés passés pour
      lesquels les charges virales seraient suffisantes et le génome
      complet non encore obtenu feront également l’objet d’un séquençage
      complet. Pour cela, un dosage préalable au Qubit 4 ou 2.0 ou au
      NanoDrop LITE sera réalisé sur les échantillons sélectionnés pour
      évaluer leur concentration (objectif : 2-100 ng/µL) et leur pureté
      (objectif : 1,8-2). Les librairies pour séquençage seront
      préparées selon les conditions du fabricant avec les kits NEBNext®
      dsDNA Fragmentase, NEBNext® Multiplex Oligos (Illumina) et Daicel
      Arbor Biosciences MYBaits®. La technologie Illumina iSeq sera
      utilisée pour le séquençage.

2.  Les données de séquençage seront utilisées pour déterminer les
      clades, et la présence ou l’absence de signatures moléculaires de
      transmission interhumaine (ex : celles laissées par l’enzyme
      humaine APOBEC3) sera recherchée dans les séquences obtenues, afin
      de déterminer la part des infections d’origine zoonotique et celle
      des infections interhumaines parmi les infections en RCA.

3.  Les séquences virales obtenues et leurs données épidémiologiques
      associées seront partagées dans une base de données commune avec
      les autres instituts du PN déclinant le projet VAXPOX, permettant
      de renforcer la surveillance génomique collaborative entre les
      instituts du réseau. Un modèle phylodynamique sera développé en
      utilisant l’ensemble des séquences disponibles dans les banques de
      données publiques, afin de décrire la dynamique de transmission
      zoonotique et interhumaine.

WP4 : Modélisation mathématique

Pour répondre aux objectifs secondaires « modéliser la circulation du
mpox en RCA sur la base des données collectées » et « identifier les
stratégies vaccinales les plus efficientes dans le contexte en RCA »,
les données collectées dans les autres WP seront utilisées et complétées
avec les données issues de la surveillance nationale. Des modèles
mathématiques décrivant la circulation du mpox en RCA seront développés
et serviront à évaluer différentes stratégies de vaccination pour
identifier les stratégies les plus efficientes.

Les étapes qui seront entreprises dans ce WP sont les suivantes :

1.  Un modèle mathématique décrivant la propagation du mpox sera généré
      en prenant en compte les différentes voies de transmission du
      MPXV, y compris la transmission interhumaine sexuelle ou non
      sexuelle, ainsi que la transmission zoonotique. Le déclin attendu
      des titres d’anticorps à distance de l’infection sera pris en
      compte dans l’analyse des résultats des enquêtes sérologiques. La
      population modélisée sera structurée par âge et par groupe à
      risque.

2.  Un échantillonnage par chaîne de Markov Monte Carlo sera utilisé
      pour calibrer le modèle aux données concernant les cas confirmés
      de mpox en RCA collectées par le système national de surveillance.
      Cette calibration permettra de quantifier la contribution des
      différentes voies de transmission à la propagation globale.

3.  Le modèle obtenu servira à simuler des épidémies selon différentes
      stratégies de vaccination. La charge infectieuse évitée par chaque
      stratégie sera estimée, et celle ayant l’impact maximal sera
      identifiée. Le modèle sera calibré aux données de RCA provenant de
      contextes variés (urbain vs. rural) afin de déterminer les
      spécificités propres à chaque contexte, et d’évaluer si ces
      différences justifient l’application de stratégies vaccinales
      différenciées.

4.  Des analyses coût-efficacité des stratégies de vaccination
      envisageables en RCA seront réalisées, en prenant en compte les
      différences de gravité de la maladie selon certaines
      caractéristiques individuelles, telles que l’âge ou le clade
      viral. Le fardeau de la maladie évité estimé par le modèle de
      transmission sera converti en mesures économiques ou sanitaires.
      Cette harmonisation des résultats du modèle permettra de calculer
      les ratios coût-efficacité incrémentaux et d’offrir aux décideurs
      des informations sur la valeur des différentes stratégies
      vaccinales. Des analyses de sensibilité seront menées pour évaluer
      la transférabilité des résultats à d'autres contextes
      épidémiologiques.

Population étudiée

2.  

Description des populations étudiées

-   Population générale en zone rurale : individus consultant à
      l’hôpital de Mbaiki (préfecture de la Lobaye, 100 km au Sud de
      Bangui). La région de la Lobaye est une région rapportant
      régulièrement des cas confirmés de mpox, avec suspicion de
      transmission zoonotique.

-   Professionnels de santé du Centre hospitalier et universitaire
      communautaire de Bangui (CHUCB).

-   Groupes particulièrement à risque de transmission du virus par voie

      au Centre National de Référence des Infections Sexuellement
      Transmissibles et de la Thérapie Antirétrovirale (CNRISTAR) de
      Bangui.

Ces trois groupes ciblés par les enquêtes de séroprévalence sont
complémentaires en termes d’expositions potentielles au mpox (exposition
zoonotique pour la population générale à Mbaiki, exposition
professionnelle pour les professionnels de santé du CHUCB, exposition
sexuelle pour les travailleurs du sexe et individus porteurs d’IST). Le
projet VAXPOX-RCA permettra d’estimer la circulation passée du MPXV
parmi ces groupes, et d’évaluer leur acceptabilité vis-à-vis du vaccin
mpox qui pourrait leur être proposé dans le cadre de campagnes de
vaccination.

-   Membres des ménages de cas confirmés de mpox identifiés dans le
    cadre de la surveillance nationale. L’utilisation des échantillons
    collectés au sein des ménages de cas confirmés permettra de mieux
    évaluer le taux de transmission du virus MPXV au sein des ménages,
    ce qui est un paramètre clé pour les modèles mathématiques qui
    modéliseront la circulation du virus en RCA.

-   Cas confirmés de mpox détectés dans le cadre du système de
    surveillance nationale. L’analyse des séquences virales isolées des
    cas confirmés de mpox permettra notamment d’évaluer les parts
    respectives de la transmission zoonotique et interhumaine parmi les
    cas confirmés en RCA.

Justification du nombre de sujets à inclure dans la recherche

WP1-2 Epidémiologie et sérologie

Objectif principal

Pour estimer la séroprévalence des anticorps anti-MPXV par âge, sexe et
groupe à risque dans différentes populations urbaines et rurales en RCA,
900 sujets seront recrutés au sein de groupes représentatifs de la
population générale et de populations particulièrement à risque de
transmission du virus par voie sexuelle. Ces sujets seront répartis
comme suit :

-   300 sujets en zone rurale recrutés au sein de l’hôpital de Mbaiki
      (région de la Lobaye),

-   200 sujets recrutés parmi les professionnels de santé du CHUCB,

-   200 travailleurs du sexe identifiés sur des sites de transaction
      sexuelle à Bangui et adressés au CNRISTAR de Bangui,

-   200 patients atteints d’IST recrutés au CNRISTAR de Bangui.

Justification :

-   Pour les groupes particulièrement à risque de transmission du virus
      par voie sexuelle et le groupe des professionnels de santé,
      homogènes en termes de catégorie d’âge, 200 participants seront
      recrutés : avec 200 participants, les intervalles de confiance
      exacts binomiaux à 95% pour les estimations de séroprévalence de
      3%, 10% et 50% seront respectivement de (1.1-6.4%), (6.2-15.0%),
      et (42.9-57.1%).

-   Pour les populations en zones rurales, les participants recrutés
    seront plus hétérogènes en termes de catégories d’âge. Le nombre de
    sujets recrutés par région sera donc porté à 300 pour permettre une
    puissance statistique suffisante par catégorie d’âge : avec 300
    participants les intervalles de confiance exacts binomiaux à 95%
    pour les estimations de séroprévalence de 3%, 10% et 50% seront
    respectivement de (1.4-5.6%), (6.8-14.0%), et (44.2-55.8%).

Objectifs secondaires

Pour « estimer le taux de transmission du MPXV au sein de ménages en
RCA », les échantillons collectés dans le cadre des activités de
surveillance auprès des membres des ménages de cas confirmés de mpox
seront utilisés. Si on estime à 70 le nombre de cas confirmés de mpox
qui surviendraient en RCA au cours des 2 ans de la recherche, et à
raison d’une moyenne de 10 sujets prélevés par foyer de cas confirmés,
le nombre total d’échantillons qui seraient utilisés pour répondre à cet
objectif secondaire peut être estimé à 700 échantillons.

Pour répondre à l’objectif secondaire de la recherche « quantifier les

transmission interhumaine du MPXV dans la dissémination du mpox en

 Description des zones d’intervention

Préfecture de la Lobaye

La préfecture de la Lobaye est située dans la partie sud de la
République Centrafricaine. Elle est bordée au Nord par la préfecture de
la Sangha-Mbaéré, à l’Ouest par le Congo, au Sud par la République
Démocratique du Congo et à l’Est par la Préfecture de l'Ombella-M'Poko.
Elle se trouve à une altitude de 580 mètres. Cette région est définie
par un climat équatorial, comprenant une saison des pluies qui s'étend
d'avril à novembre et une saison sèche de décembre à mars. La pluie se
répartit sur huit mois et varie de 1200 à 1600 mm par an. Sa population
est estimée à 221.000 habitants.

Historiquement, la région de la Lobaye a rapporté de nombreux cas de
mpox, principalement liés à des transmissions d’origine zoonotique. Le
chef-lieu de la préfecture de la Lobaye est Mbaiki, disposant d’un
centre de traitement pour les patients mpox au sein de l’hôpital
préfectoral. De plus, cette zone est facile d’accès depuis Bangui,
permettant l’investigation des cas suspects ainsi que la conduite de
recherches par l’IPB.

Critères d’inclusion

Sujets représentatifs de la population générale en zone rurale

-   Individu âgé de 1 an ou plus,

-   consultant à l’hôpital de Mbaiki,

-   ayant consenti ou dont le responsable légal a consenti à sa

-   Individu âgé de 18 ans ou plus,

-   travaillant en tant que professionnel de santé du Centre Hospitalier
    et Universitaire Communautaire de Bangui (CHUCB),

-   ayant consenti à participer à l’enquête de séroprévalence
    (questionnaire et prélèvement de sang veineux).

Groupes particulièrement à risque de transmission du virus par voie sexuelle 

Travailleurs du sexe :

-   Individu âgé de 18 ans ou plus,

-   exerçant la profession de travailleur du sexe depuis au moins 6
    mois,

-   fréquentant les endroits de transaction sexuelle à Bangui,

-   adressé par les pairs éducateurs au CNRISTAR de Bangui,

-   ayant consenti à participer à l’enquête de séroprévalence
    (questionnaire et prélèvement de sang veineux).

Patients IST :

-   Individu âgé de 18 ans ou plus,

-   consultant au CNRISTAR de Bangui,

-   ayant consenti à participer à l’enquête de séroprévalence

-   Prélèvements de pus, croûte, sang ou gorge positifs au MPXV,

Echantillons des membres des ménages d’individus infectés par le MPXV

-   Prélèvements de sang,

-   issus d’individus faisant partie du ménage d’un cas confirmé mpox,

-   présents au sein du foyer lors du passage de l’équipe
    d’investigation dans le cadre de la surveillance nationale du mpox.

Critères de non-inclusion

Critères de non-inclusion de sujets :

-   Individu dont l'état de santé n'est pas compatible avec un
    prélèvement sanguin selon l'investigateur,

-   Individu sous mesure de protection juridique,

-   Individu hors d’état de consentir,

-   Individu nécessitant une prise en charge médicale immédiate.

Critères de non-inclusion d’échantillons :

pour procéder au séquençage complet du génome viral.

Modalités spécifiques aux enquêtes sérologiques auprès de populations ciblées (WP1-2)

Etapes en amont de la recherche

En amont de la recherche, un partenariat sera mis en place avec les
différents sites d’investigation :

a)  A Bangui : Centre National de Référence des Infections Sexuellement
    Transmissibles et de la Thérapie Antirétrovirale (CNRISTAR), et
    Centre Hospitalier et Universitaire Communautaire de Bangui
    (CHUCB) ;

b)  A Mbaiki : Hôpital préfectoral de Mbaiki.

L’objectif est d’assurer une collaboration fluide entre l’équipe
d’épidémiologie de l’IPB et l’équipe d’investigation locale pour le bon
déroulement de l’étude.

1.  Prise de contact : Une prise de contact officielle avec les deux
    hôpitaux ciblés (CHUCB et Mbaïki) et le CNRISTAR de Bangui se
    matérialisera par la rencontre entre les responsables du projet
    (équipe d’épidémiologie de l’IPB) et la direction de chaque
    structure de santé pour leur présenter les objectifs de l’étude, le
    rôle attendu de la structure de santé, les bénéfices attendus pour
    la communauté, et discuter de la faisabilité et des ressources
    disponibles sur place (personnel, équipements de stockage, espace
    pour prélèvements).

2.  Désignation du point focal : Un médecin ou un responsable désigné
    par chaque structure servira de point focal pour assurer la
    coordination quotidienne avec l’équipe d’investigation.

3.  Une fois le partenariat établi, médecins, infirmiers et agents de
    santé qui constitueront l’équipe d’investigation locale seront
    formés par l’équipe d’épidémiologie de l’IPB sur:

-   Les objectifs du projet ;

-   Les critères d’inclusion ;

-   Le processus d’information et de consentement ;

-   Les aspects éthiques et la confidentialité ;

-   Le processus de prélèvement et de documentation.

4.  Préparation logistique sur les sites : Aménagement d’un espace pour
    les entretiens et prélèvements, mise à disposition du matériel
    nécessaire (kits de prélèvement, fiches, équipements de
    conservation, etc.), vérification de la capacité de conservation
    (chaîne du froid locale).

5.  Signature d’un accord de collaboration : Document formel
    (convention) signé entre les parties, définissant les rôles,
    responsabilités et modalités de coordination.

Modalités spécifiques aux travailleurs du sexe : Contrairement aux
autres participants recrutés dans le cadre de cette recherche, les
travailleurs du sexe seront approchés en-dehors du cadre du soin, dans
des lieux de transaction sexuelle identifiés à Bangui (voir Figure 2,
section 5). Les individus intéressés à participer seront adressés au
CNRISTAR pour le recrutement et les procédures de l’étude. Afin
d’assurer le bon déroulement de cette démarche, l’équipe d’épidémiologie
de l’IPB, appuyée par l’équipe du Professeur Longo au sein du Ministère
de la Santé et des Populations, prendra contact avant le début de la
recherche avec les autorités administratives et policières de la ville
afin d’expliquer le bien-fondé de l’enquête. Aussi, l’implication de
l’Association des Filles Libres (travailleuses du sexe) et l’Association
Alternatives-Centrafrique (travailleurs du sexe), travaillant
habituellement avec le Ministère de la Santé et de la Population pour ce
type d’étude, sera sollicitée afin de proposer des pairs éducateurs qui
seront formés par l’équipe de l’IPB à la présentation de l’étude VAXPOX.

Recrutement des participants

Les modalités de recrutement des participants seront adaptées à chaque
contexte, dans le respect des principes de confidentialité et de
non-stigmatisation, notamment pour les populations exposées à des
risques de discrimination. Une attention particulière sera portée à la
formulation des critères d’inclusion, à la langue utilisée, et à la
nature du consentement éclairé. A noter que toute personne présentant
des lésions évocatrices d’une infection en cours par le MPXV sera
référée immédiatement selon le circuit clinique national ; la
participation à l’enquête restera possible.

-   Enquête auprès d’individus en zone rurale : patients de l’hôpital de
      Mbaïki

Le partenariat entre l’unité d’épidémiologie de l’IPB et l’hôpital de
Mbaiki permettra de sensibiliser les médecins et infirmiers de l’hôpital
pour proposer l’étude aux patients qui viennent en consultation. Les
médecins et les infirmiers de l’hôpital auront à : expliquer le projet
(via la note d’information en français et en langue nationale),
recueillir le consentement éclairé, administrer le questionnaire
standardisé (pour collecter des informations socio-démographiques et
données médicales) et réaliser le prélèvement sanguin.

-   Enquête auprès de professionnels de santé du Centre Hospitalier
      Universitaire et Communautaire de Bangui

La collaboration entre l’unité d’épidémiologie de l’IPB et le CHUCB
permettra de mettre en place un plan d’action pour le recrutement dans
les différents services de l’établissement, qui consistera en :

-   La sensibilisation des médecins, infirmiers, agents de santé pour
    participer à l’enquête à travers un réseau de correspondants dans
    chaque service. Un référent local sera identifié dans chaque service
    par le chef ou le responsable de service (ex : cadre, médecin
    référent, IDE).

-   La formation de ces référents pour qu’ils assurent la communication
    auprès de leurs collègues de service et leur remettre des flyers
    présentant l’étude.

-   L’organisation de réunions régulières de suivi avec les référents
    pour faire le point sur les éventuelles difficultés avant
    recrutement, puis le recrutement.

-   L’établissement d’un calendrier précis par service (début du
      recrutement, points d’étape) pour rendre le circuit des
      participants plus fluide et ne pas les surcharger.

Les professionnels de santé du CHUCB qui souhaiteront participer seront
orientés vers les équipes du Service d’Infectiologie qui auront à
expliquer le projet, recueillir le consentement éclairé, administrer le
questionnaire et réaliser le prélèvement sanguin.

-   Enquête auprès de travailleurs du sexe à Bangui

Le recrutement des travailleurs du sexe sera réalisé au travers de pairs
éducateurs de l’Association les Filles Libres et de l’association
Alternatives-Centrafrique. Ces pairs éducateurs vont approcher les
individus éligibles sur la base du volontariat, au sein des lieux
identifiés de transaction sexuelle à Bangui (Figure 2, section 5). Ces
pairs éducateurs expliqueront oralement les objectifs de l'enquête et
éventuellement organiseront des séances de sensibilisation collective.

Les travailleurs du sexe intéressés à participer suite à cette étape
d’explication orale seront invités à se rendre au CNRISTAR de Bangui
pour participer à l’enquête (information, consentement éclairé, recueil
d’information sur questionnaire et prélèvement). Les 200 premiers
travailleurs du sexe à se présenter au CNRISTAR seront recrutés,
indépendamment de leur sexe.

-   Enquête auprès de patients du Centre National de Référence des IST
      et de la Thérapie Antirétrovirale (CNRISTAR) de Bangui

L’étude sera proposée aux patients IST se présentant au CNRISTAR lors
des consultations assurées par les médecins de la structure. Les
individus intéressés pour participer seront adressés à l’équipe
d’investigation locale composée d’un médecin et d’un infirmier du
CNRISTAR. Ils auront pour rôle d’expliquer l’étude, recueillir le
consentement éclairé, administrer le questionnaire et effectuer le
prélèvement de sang.

Procédures de l’étude, circuits des données et échantillons

Information et inclusion dans l’étude : Sur chaque site investigateur,
les participants potentiels identifiés recevront une note d’information
décrivant l’enquête de séroprévalence, le consentement éclairé écrit
sera recueilli, et un code participant unique sera attribué. Pour les
mineurs recrutés au sein de l’hôpital de Mbaiki, le consentement éclairé
d’un responsable légal sera recueilli, et l’assentiment du mineur à
partir de 7 ans sera recherché. Un participant éligible peut donner son
consentement immédiatement ou revenir ultérieurement s’il souhaite
réfléchir.

Questionnaire et prélèvement sanguin : Sur chaque site investigateur, un
questionnaire standardisé (CRF papier avec codification des
participants) sera administré par l’équipe investigatrice pour collecter
des informations individuelles (données socio-démographiques et données
médicales) et un prélèvement de sang sera effectué par l’infirmier de
l’étude (ou préleveur) : 2.5 ml de sang veineux pour les participants de
6 ans ou plus, jusqu’à 500 µl de sang capillaire pour les participants
de 1 à 5 ans. Le prélèvement sera étiqueté avec le code participant.

Compensation : En compensation du temps passé pour les procédures de
l’étude, un forfait transport d’une valeur de 2.000 Francs CFA sera
proposé aux participants adultes recrutés à l’hôpital préfectoral de
Mbaiki, au CHUCB, et les patients IST recrutés au CNRISTAR de Bangui.
Pour les travailleurs du sexe adressés par les pairs éducateurs au
CNRISTAR de Bangui, une compensation de 6.000 Francs CFA sera proposée.
Pour les participants mineurs recrutés à l’hôpital préfectoral de
Mbaiki, un savon sera proposé en compensation.

Gestion des données : Sur chaque site investigateur, les formulaires
papiers (consentement et CRF) seront centralisés par l’équipe
d’investigation locale. Les CRF papier seront transmis à l’équipe
centrale (équipe d’épidémiologie IPB) à Bangui en même temps que les
échantillons. Les formulaires de consentement seront conservés sur les
sites investigateurs sous armoire sécurisée pendant toute la période des
recrutements, puis transmis sous enveloppe scellée à l’IPB à l’issue des
recrutements.

A l’IPB, les CRF papier seront centralisés par l’équipe d’épidémiologie
de l’IPB (attaché de recherche clinique (ARC) en charge). L’ARC
vérifiera les CRF papier : complétude des données (toutes les cases
nécessaires remplies) ; signature de l'investigateur si nécessaire ;
vérifiera que les données sont lisibles, cohérentes, et sans erreur
manifeste. Les données des CRF papier seront saisies en double saisie
dans la base de données électronique du projet par les équipes de l’IPB.

Gestion des échantillons biologiques : Sur chaque site investigateur,
les échantillons seront étiquetés (avec le code défini pour chaque
participant), sécurisés et conservés temporairement dans des conditions
appropriées (réfrigération). Les prélèvements effectués sur les sites de
recrutement à Bangui seront transportés quotidiennement vers l’IPB en
respectant la chaîne du froid. Les prélèvements effectués à l’hôpital
préfectoral de Mbaiki feront l’objet d’un pré-traitement par les équipes
de l’hôpital (centrifugation pour récupération des sérums). Les sérums
seront conservés dans des conditions appropriées et transférés à l’IPB
tous les 3 à 4 jours.

Les prélèvements issus des différents sites de recrutement seront reçus
à l’IPB par l’équipe d’épidémiologie (ARC en charge). L’ARC vérifiera la
conformité des échantillons biologiques reçus : bon état de conservation
(température, conditionnement, intégrité des tubes) ; étiquetage clair
et identifiant codé des participants (pas de données nominatives) ;
concordance avec le bordereau d’envoi.

Modalités spécifiques aux échantillons et données issus de la surveillance nationale du mpox

Ce projet de recherche vient s’ajouter à la procédure de surveillance
nationale de la variole du singe effectuée par le Ministère de la Santé
de RCA, l’OMS et l’Institut Pasteur de Bangui. Chaque épidémie fait
l’objet d’une investigation des cas humains de variole du singe avec
recherche des contacts humains et du réservoir animal, à travers la
collecte d’échantillons biologiques et de données personnelles sur le
formulaire d’investigation d’épidémie de RCA.

Lors de la déclaration d’un cas suspect de variole du singe dans un
poste de santé, un prélèvement biologique est envoyé au laboratoire des
arbovirus, des fièvres hémorragiques et des virus émergents de l’IP de
Bangui pour réaliser une qPCR. Si la qPCR MPXV est positive, et que le
cas confirmé se trouve dans une zone géographique accessible et
sécurisée, une équipe d’investigation des épidémies de variole du singe
est envoyée sur place pour l’enquête autour du cas confirmé :
prélèvement biologique des autres cas éventuels, des contacts des cas,
et administration d’un questionnaire pour le recueil des données
sociodémographiques et épidémiologiques.

L’équipe vérifie également que le diagnostic soit fait chez tous les cas
suspects et que l’isolement en chambre seule (afin d’éviter la
transmission du virus) et la prise en charge symptomatique des patients
soient satisfaisants. Les analyses biologiques sont réalisées à l’IP de
Bangui et un retour de résultats individuels auprès des participants est
fait par l’intermédiaire du personnel du poste de santé de la zone.

Les échantillons de sang prélevés dans le cadre de ces activités de
surveillance au sein de ménages de cas confirmés de mpox seront analysés
pour détecter des anticorps spécifiques anti-MPXV. Ceci permettra
d’identifier tous les cas secondaires au sein des ménages de cas
confirmés, afin de mieux définir le taux de transmission du MPXV au sein
des ménages. Dans la mesure du possible, ces prélèvements seront
effectués au moins 6 semaines après l’apparition des symptômes du
dernier cas confirmé par PCR dans le foyer de ce cas. Ce délai d’au
moins 6 semaines permettra aux anticorps spécifiques anti-MPXV d’être
détectables pour tous les cas secondaires au sein du foyer. Les
personnes concernées pourront être informées de la recherche VAXPOX dans
le cadre des activités de surveillance.

Modalités spécifiques aux activités de virologie moléculaire (WP3)

Depuis 2002, l’IPB a confirmé 298 cas de mpox dans le cadre de son
mandat de LNR pour le diagnostic du mpox (effectif en date de décembre
2025). Les souches virales isolées des prélèvements de ces cas positifs
disponibles en biobanque à l’IPB et qui n’ont pas encore bénéficié d’un
séquençage du génome complet, ainsi que toute souche virale qui serait
isolée de nouveaux cas confirmés par l’IPB pendant la durée de la
recherche, seront incluses dans les analyses génomiques décrites dans la
section « 4. Méthodologie » (voir sous-section 4.2.2).

Les personnes concernées pourront être informées de la recherche VAXPOX
dans le cadre des activités de surveillance.

Précisions sur les analyses sérologiques

Les échantillons de sérum seront analysés à l’IPB avec un test
sérologique multiplex à base de microsphères (MMIA) utilisant la
technologie Luminex. Ce test a été développé à l’IPP (et transmis à
l’IPB) pour la détection simultanée des anticorps IgM ou IgG dirigés
contre plus de 35 antigènes du MPXV et des protéines apparentées des
orthopoxvirus (Vaccinia, Camelpox, Mousepox / virus de l’Ectromélie). La
détection des anticorps dirigés contre d’autres orthopoxvirus permet de
contrôler la spécificité des anticorps détectés pour les antigènes du
MPXV. Ce test sérologique s’est avéré efficace pour détecter des
anticorps spécifiques du MPXV dans des sérums prélevés chez des patients
hospitalisés à Paris lors de l’épidémie de MPXV clade II en 2022, ainsi
que chez des cas confirmés de mpox lors des épidémies de MPXV clade Ia
en RCA entre 2021 et 2023 (AUC de 97 % et 96 %, respectivement, données
non publiées). Cette analyse permettra : (1) d’établir une estimation de
la séroprévalence des anticorps anti-MPXV par âge et sexe dans les
différentes populations ciblées dans cette recherche ; (2) d’éclairer la
modélisation de paramètres épidémiologiques clés pour le WP4, tels que
le taux de transmission du MPXV au sein des ménages.

Au-delà d’antigènes du MPXV, le test comprendra des antigènes permettant
de mesurer des anticorps dirigés contre des agents pathogènes
responsables de maladies infectieuses d’intérêt pour la santé publique,
notamment des maladies évitables par la vaccination (diphtérie, tétanos,
Haemophilus influenza type b, coqueluche, rotavirus, méningite A,
rougeole, rubéole, fièvre jaune), des maladies respiratoires (grippe,
adénovirus, coronavirus), des arboviroses (dengue, Zika, chikungunya),
des parasites du paludisme, les herpesvirus, le typhus des broussailles,
le choléra, Borrelia crocidurae, la leptospirose, et des pathogènes
responsables de maladies tropicales négligées (Schistosoma mansoni,
Chlamydia trachomatis, Wucheria bancrofti, Onchocerca volvulus,
Cryptospiridium parvum, Giardia intestinalis, Strongyloides stercoralis,
Entamoeba histolytica).

Cette analyse permettra d’évaluer la séroprévalence des anticorps pour
un panel de pathogènes d’intérêt dans les populations ciblées dans cette
recherche. Aussi, cela permettra de détecter et corriger les
interférences potentielles entre les anticorps dirigés contre différents
pathogènes présents localement, et les antigènes MPXV utilisés dans le
test. Enfin, ces analyses sérologiques témoigneront de l’exposition
passée à différents pathogènes, elles ne constitueront pas un diagnostic
d’infection en cours. Aucun résultat incident n’est donc à attendre de
ces analyses.

Contrôle des résultats de sérologie

Un sous-groupe d’échantillons pourra être envoyé à l’IPP pour
vérification des résultats par les équipes ayant mis au point le test
sérologique utilisé dans le cadre du projet. L’envoi d’échantillons
serait alors tracé à l’IPB et à l’IPP. Les échantillons resteront la
propriété de l’IPB et ne pourront être utilisés ou détruits sans
autorisation. Les analyses s’effectueront exclusivement par le personnel
de l’IPP autorisé impliqué dans le projet (unités ERI ou IDEA). Les
participants auront la possibilité de cocher « oui » ou « non » dans le
document de consentement concernant cette possibilité d’envoi des
échantillons à l’IPP.

Conservation et réutilisation des échantillons

Les échantillons stockés à l’IPB et les éventuels échantillons envoyés à
l’IPP restant la propriété de l’IPB seront conservés par chaque
institution dans des locaux sécurisés pendant toute la durée du projet +
10 ans (soit 12 ans au total). La conservation des échantillons au-delà
de cette durée devra être autorisée avec l'accord préalable explicite du
Comité d’Éthique et Scientifique (CES) de la RCA.

Concernant la réutilisation des échantillons pour des recherches
futures, les participants auront la possibilité de cocher « oui » ou
« non » à cette modalité sur le formulaire de consentement. Pour les
participants qui auront coché « oui », l'accord préalable du CES de la
RCA devra être obtenu pour toute nouvelle recherche portant sur ces
échantillons. L’utilisation de ces échantillons par des chercheurs non
mentionnés dans ce document devra être préalablement approuvée par
l’IPB. Les participants qui auront coché « non » peuvent tout de même
participer à la recherche VAXPOX-RCA, mais leurs échantillons seront
détruits à l’issue de la complétion des objectifs du protocole.

identifiant et d’un mot de passe personnels, et permet de différencier
les droits d’accès (modification, lecture, extraction...) aux
formulaires et aux données qu’ils contiennent selon le profil des
utilisateurs (investigateurs, équipes d’investigations, gestionnaires de
données, attachés de recherche clinique).

Un questionnaire au format papier (CRF) reprenant les formulaires créés
sur la base de données électronique sera administré aux participants par
les équipes d’investigation. Les données seront saisies à posteriori
(double saisie) à l’IPB dans la base de données électronique.

données épidémiologiques associées seront effectuées conjointement par
les équipes de l’IPB du service d’Epidémiologie, du Laboratoire des
arbovirus, fièvres hémorragiques virales, virus émergents et zoonoses et
du Service de Plateforme Surveillance Génomique, avec l’appui de
plusieurs équipes de l’IPP : les unités Epidémiologie des maladies
émergentes (UEME), Environnement et risques infectieux (ERI), et
Epidémiologie et analyse des maladies infectieuses (IDEA).

Les caractéristiques des sujets séropositifs et séronégatifs pour les
anticorps spécifiques d’un pathogène donné, dont le MPXV, seront
décrites par les moyennes et déviations standards, ou les médianes et
écarts interquartiles pour les variables continues, et par les
proportions avec intervalles de confiance à 95% pour les variables
discrètes. Le test du Chi-2 (ou test exact de Fisher selon les effectifs
observés) sera utilisé pour comparer les variables discrètes. Le test de
Student (ou test de Mann-Whitney selon les effectifs observés) sera
utilisé pour comparer les variables continues. Toutes les variables
associées en analyse univariée à un seuil ≤ 0,25 seront intégrées dans
une régression logistique pas à pas descendant afin de déterminer les
facteurs de risque de séroconversion. Ces facteurs de risque seront
mesurés par l’intermédiaire des Odds ratio (OR) et leur intervalle de
confiance à 95%. Le degré de signification statistique retenu sera fixé
à p<0,05.

Les données sérologiques mpox obtenues et les données épidémiologiques
associées seront partagées dans une base de données commune avec les
autres instituts du PN déclinant le projet VAXPOX, permettant d’élargir
la surveillance collaborative entre les instituts du réseau à un volet
sérologique.

Virologie moléculaire (WP3)

L’assemblage des séquences virales brutes afin de générer les génomes
entiers des souches MPXV issues des prélèvements positifs, ainsi que les
analyses phylodynamiques intégrant les métadonnées associées seront
réalisées par les équipes de l’IPB du Laboratoire des arbovirus, fièvres
hémorragiques virales, virus émergents et zoonoses et du Service de
Plateforme Surveillance Génomique, avec l’appui des unités Epidémiologie
et Physiopathologie des virus oncogènes (EPVO) et Dynamiques évolutives
des maladies infectieuses (EDID) de l’IPP.

Les séquences et métadonnées associées seront déposées sur la plateforme
Owey (serveurs IPP) en suivant le format de pathoplexus
(https://pathoplexus.org/) dans lequel les séquences et les métadonnées
sont liées, et ainsi facilement déployées pour des analyses ou le
partage. Les données génomiques qui seront obtenues seront comparées
avec les autres génomes disponibles dans GenBank sur la plateforme Owey
(https://dataset.owey.io). Les analyses combinées des données de
phylogénie et des données épidémiologiques seront réalisées grâce à une
pipeline Nextstrain³⁹. Un modèle phylodynamique sera développé via la
plateforme BEAST2⁴⁰, utilisant toutes les séquences disponibles pour
décrire les dynamiques de transmissions zoonotique et interhumaine.

Modélisation mathématique (WP4)

Les données collectées et générées dans les autres WP seront utilisées
pour estimer des paramètres clés nécessaires au développement de modèles
mathématiques décrivant la circulation du mpox en RCA. La population
modélisée sera structurée par âge et par groupe à risque d’infection. Le
modèle prendra en compte les différentes voies de transmission, y
compris la transmission interhumaine sexuelle ou non sexuelle, ainsi que
la transmission zoonotique. Un échantillonnage par chaîne de Markov
Monte Carlo sera utilisé pour calibrer le modèle aux données
disponibles. Grâce à cette calibration, la contribution des différentes
voies de transmission à la propagation globale sera quantifiée. Le
modèle obtenu sera utilisé pour simuler des épidémies selon différentes
stratégies de vaccination. Le fardeau d’infection évité par chaque
stratégie sera estimé, et celle qui maximise l’impact de la vaccination
sera identifiée. Le modèle sera calibré aux données recueillies dans
différents contextes (urbain vs rural) afin de déterminer les
caractéristiques spécifiques à chaque contexte, et d’évaluer si les
différences entre contextes justifient l’application de stratégies de
vaccination distinctes.

Des analyses coût-efficacité des stratégies de vaccination seront
réalisées, en tenant compte des différences de gravité de la maladie
selon les caractéristiques individuelles telles que l’âge ou le clade
viral. La charge d’infection évitée estimée par le modèle de
transmission sera convertie en mesures économiques ou sanitaires telles
que les ressources de santé utilisées ou les années de vie ajustées sur
la qualité. Cette harmonisation des résultats du modèle permettra de
calculer les ratios coût-efficacité incrémentaux et de fournir des
informations sur la valeur des stratégies de vaccination.

Ces travaux de modélisation seront effectués à l’IPP au sein des unités
Modélisation mathématique des maladies infectieuses (MMID) et
Epidémiologie des maladies émergentes (UEME), en collaboration avec le
Service d’épidémiologie de l’IPB.

Les équipes de recherche de l’IPB interviennent régulièrement auprès des

que l’information délivrée évite toute forme de stigmatisation des

Diffusion des résultats auprès des autorités

Les résultats du projet VAXPOX-RCA feront l’objet d’une communication
officielle au ministère de la Santé de RCA ainsi qu’aux directions
régionales et districts sanitaires des zones concernées par l’étude. Les
supports utilisés pourront être des cartographies des épidémies, des
rapports faisant ressortir les groupes particulièrement à risque mis en
évidence, et des rapports détaillant les stratégies vaccinales
identifiées comme étant les plus efficientes par les analyses
coût-efficacité. La diffusion des résultats de ce projet s’attachera à
ce que l’information délivrée évite toute forme de stigmatisation des
populations ou individus atteints. En particulier, les résultats
concernant les populations à risque de transmission d’infections par
voie sexuelle seront transmis au Programme National de Lutte contre les
IST. Le programme de lutte contre les IST est axé sur la prévention, le
dépistage gratuit lors des activités de proximité dans les communautés
et la prise en charge médicale des maladies sexuellement transmissibles.
La coordination du programme est calquée sur le système de santé de la
RCA qui est structuré en trois niveaux : central, intermédiaire,
opérationnel/districts. Il implique des acteurs communautaires à travers
des campagnes de sensibilisation et la formation d'éducateurs pairs (ex.
sensibilisation des conducteurs de taxis/camions, travailleurs de sexe,
éducation sexuelle pour les adolescents, etc.). La coordination du
programme de lutte contre les IST est donc habituée à travailler avec
les populations incluses dans l’enquête et à traiter le type
d’informations contenues dans les résultats. L’équipe de recherche
s’engage à ne pas diffuser les résultats de l’étude à d’autres acteurs
susceptibles de sur-vulnérabiliser les populations à risque de
transmission d'infections par voie sexuelle.

Diffusion des résultats auprès de la communauté scientifique

Les résultats du projet VAXPOX-RCA pourront faire l'objet de
présentations lors de réunions, ou congrès scientifiques ou médicaux,
par les membres des équipes de l’IPB et de l’IPP impliqués dans ce
projet, après validation par les co-responsables scientifiques du
projet. Les résultats feront l'objet de soumissions d’articles
scientifiques dans des revues à comité de lecture. La liste des
signataires et des personnes remerciées dans les articles scientifiques
soumis sera établie après discussion collégiale entre les responsables
scientifiques de ce projet, ainsi que les parties prenantes des WP
concernés. L’ordre des co-auteurs sera discuté en fonction de la
participation de chacun à cette recherche, participation comprenant de
façon chronologique mais de façon non exhaustive : la rédaction du
protocole, la mise en place et le suivi réglementaire et logistique de
la recherche, la collecte des données et échantillons biologiques dans
le cadre des activités de recherche, les analyses de ces échantillons et
données, leur interprétation, et la rédaction des articles
scientifiques.

Définitions

Événements indésirables

Un événement indésirable est défini comme toute manifestation nocive
chez un participant à une étude clinique, et qui n'est pas
nécessairement liée à la participation dans l’étude.

Événements / effets indésirables graves (EIG)

Un événement indésirable doit être considéré comme grave dès lors
qu’il : entraîne le décès, met en jeu le pronostic vital, entraîne une
incapacité ou une invalidité temporaire ou définitive, nécessite ou
prolonge une hospitalisation du patient, entraîne une anomalie
congénitale ou néonatale, est médicalement important (ce qui signifie :
nécessite une prise en charge pour éviter l’aggravation vers un des
stades ci-dessus).

Les événements suivants ne seront pas considérés comme EIG :

-   décès du fait de la progression d’une pathologie concomitante du
    participant (le cas échéant),

-   hospitalisations d’une durée inférieure à 24 heures (i.e. ne
    nécessitant pas une nuitée à l’hôpital),

-   hospitalisations qui auraient été planifiées avant la participation
    du sujet au protocole,

-   hospitalisations pour la prise en charge d’une pathologie
    concomitante diagnostiquée avant la participation du sujet au
    protocole, excepté si l’intensité ou la fréquence de celle-ci est
    augmentée depuis la participation du sujet au protocole,

-   hospitalisations pour des procédures cliniques de routine ou pour
    des raisons sociales.

Effets Indésirables attendus

Un effet indésirable attendu est un effet prévisible, lié aux actes
d’examens et de prélèvements requis par le protocole (pour les
prélèvements sanguins : douleur, hématome, malaise vagal).

Risques encourus par la recherche

La nature des actes médicaux réalisés est très peu à risque de
complication. Les seuls événements indésirables attendus sont liés à la
prise de sang : légère douleur au point de ponction au moment de
l’insertion de l’aiguille, petit hématome au point de prélèvement ou
malaise vagal bénin. Ces éventuels risques physiques seront minimisés
car les gestes seront pratiqués par du personnel formé à cet effet et
réalisés avec du matériel stérile à usage unique.

Recueil, notification et suivi des évènements indésirables 

Cette étude ne comportant que des risques et des contraintes minimes,
les éventuels évènements indésirables ne feront pas l’objet de
notification au promoteur mais seulement auprès du service concerné dans
le cadre du soin et dans lequel le participant pourra être adressé pour
une prise en charge adéquate.

Assurance

L’IPB, en tant que promoteur, n’est pas tenu de souscrire un contrat
d’assurance spécifique pour cette recherche n’impliquant pas
l’administration de produits médicaux, conformément aux dispositions
légales et réglementaires centrafricaines en matière de recherche sur la
personne humaine impliquant un prélèvement sanguin.

Monitoring

S’agissant d’une recherche impliquant la personne humaine à risques et
contraintes minimes prévoyant une collecte d’échantillons selon des
actes peu ou pas invasifs, aucun suivi étroit n’est jugé nécessaire.
Néanmoins pour satisfaire aux exigences des Bonnes Pratiques Cliniques,
les visites de monitoring suivantes seront réalisées sur les sites
investigateurs.

Une réunion de pré-ouverture afin de :

- Vérifier les moyens techniques et les compétences des sites
investigateurs,

- Valider les modalités pratiques et logistiques.

- Définir, le cas échéant et en accord avec les centres, un plan
d’action en vue de la mise en place officielle de l’étude,

Une réunion d’ouverture afin de :

- Vérifier les moyens techniques et les compétences des sites
investigateurs,

- Rappeler les aspects scientifiques du protocole, la réglementation,

- Valider les modalités pratiques et logistiques.

Une visite de suivi par an, plus si nécessaire, afin de :

- Vérifier la validité des consentements,

- Vérifier ou collecter les données sur les fiches prévues à cet effet

- Vérifier la documentation des échantillons transférés.

Une visite de clôture administrative afin de :

- Vérifier la validité des consentements,

- Vérifier les fiches de recueil de données et la documentation des
échantillons transférés,

- S’assurer du contenu et de l’archivage du dossier investigateur.

tout représentant du promoteur désigné à cet effet.

personne autorisée.

GASASIRA, fournira un appui à l’IPB conjointement avec l’unité ERI de
l’IPP pour la mise en place du test, la formation des personnels, et
l’analyse des données brutes obtenues.

Unité Epidémiologie et physiopathologie des virus oncogènes (EPVO)

Le Dr Nicolas BERTHET, chercheur au sein de l’unité d’Epidémiologie et

en collaboration avec l’unité UEME en RCA. Avec la contribution de son
doctorant Alexander TENDU, Dr Nicolas BERTHET apportera son expertise
aux équipes du IPB pour les activités de virologie moléculaire.

Unité Dynamiques évolutives des maladies infectieuses (EDID)

Le Dr Sebastian DUCHENE GARZON dirige l’unité Dynamiques évolutives des
maladies infectieuses qui est spécialisée dans les méthodes
phylodynamiques appliquées aux données génomiques, utilisées pour
comprendre les zoonoses et mettre en place la surveillance génomique
d’épidémies. Avec l’appui de Jordi SEVILLA, le Dr Sebastian DUCHENE
GARZON apportera son expertise aux équipes du IPB pour mener à bien les
analyses phylogénétiques prévues dans le projet.

Hub de Bioinformatique et Biostatistique

Le Dr Laurent ESSIOUX dirige le Hub de Bioinformatique et Biostatistique
et appuiera les équipes EPVO et EDID dans la mise en place du recueil
des données génomiques virales et métadonnées associées sur la
plateforme OWEY hébergée sur les serveurs de l’IPP.

WP4 Modélisation mathématique

Unité Modélisation mathématique des maladies infectieuses (MMID)

L’unité de Modélisation mathématique des maladies infectieuses dirigée
par le Dr Simon CAUCHEMEZ développe des méthodes statistiques et
mathématiques pour mieux comprendre la manière dont les agents
pathogènes se propagent au sein des populations humaines, ainsi que
l'impact des interventions afin de soutenir la prise de décision
politique et d'optimiser les stratégies de contrôle. L’équipe du Dr
Simon CAUCHEMEZ sera responsable des activités du WP5, en particulier la
modélisation de la circulation du mpox en RCA, intégrant l’ensemble des
données recueillies et générées par les autres WP. Les méthodes et
résultats obtenus par l’équipe du Dr Simon CAUCHEMEZ seront partagés et
discutés en collaboration avec les modélisateurs du IPB.
