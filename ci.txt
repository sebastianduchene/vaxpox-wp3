RESUME SYNOPTIQUE DU PROTOCOLE

+----------------------------------+----------------------------------+
| 1.  Titre complet                | Modélisation de la circulation   |
|                                  | du mpox, identification de       |
|                                  | groupes à risque d’infection, et |
|                                  | analyse coût-efficacité de       |
|                                  | stratégies vaccinales en Côte    |
|                                  | d’Ivoire                         |
+----------------------------------+----------------------------------+
| 2.  Acronyme                     | VAXPOX-CI                        |
+----------------------------------+----------------------------------+
| 3.  Promoteur                    | Institut Pasteur de Côte         |
|                                  | d’Ivoire (IPCI)                  |
+----------------------------------+----------------------------------+
| 4.  Responsables scientifiques   | Man-Koumba SOUMAHORO-AGBO, IPCI  |
|                                  |                                  |
|                                  | Arnaud FONTANET, IPP             |
+----------------------------------+----------------------------------+
| 5.  Type d’étude                 | Etude transversale sur           |
|                                  | échantillons biologiques         |
|                                  | d’origine humaine avec données   |
|                                  | personnelles de santé, basée sur |
|                                  | une approche pluridisciplinaire  |
|                                  | comprenant 4 volets :            |
|                                  | épidémiologie, sérologie,        |
|                                  | virologie et modélisation        |
|                                  | mathématique.                    |
+----------------------------------+----------------------------------+
| 6.  Rationnel de l’étude         | En 2022 puis en 2024,            |
|                                  | l’Organisation Mondiale de la    |
|                                  | Santé (OMS) a déclaré le mpox    |
|                                  | “Urgence de Santé Publique de    |
|                                  | Portée Internationale” en raison |
|                                  | de l’émergence de sous-clades du |
|                                  | virus de la variole du singe     |
|                                  | (MPXV), de leur propagation vers |
|                                  | des régions et des pays          |
|                                  | jusque-là non touchés, y compris |
|                                  | dans des zones urbaines, et la   |
|                                  | transmission interhumaine        |
|                                  | soutenue, y compris par voie     |
|                                  | sexuelle. Des millions de doses  |
|                                  | de vaccins (formes atténuées du  |
|                                  | vaccin contre la variole         |
|                                  | conférant une protection croisée |
|                                  | contre le MPXV) vont être        |
|                                  | distribuées aux pays africains   |
|                                  | les plus touchés pour contrer    |
|                                  | l’épidémie. Il est urgent de     |
|                                  | disposer de données              |
|                                  | scientifiques pour mieux         |
|                                  | orienter les stratégies          |
|                                  | vaccinales et identifier les     |
|                                  | populations à prioriser pour la  |
|                                  | vaccination.                     |
|                                  |                                  |
|                                  | Le projet VAXPOX a pour objectif |
|                                  | d’accompagner les politiques de  |
|                                  | santé publique à mesure que les  |
|                                  | vaccins deviennent disponibles,  |
|                                  | via une approche                 |
|                                  | pluridisciplinaire associant     |
|                                  | épidémiologie, sérologie,        |
|                                  | virologie moléculaire et         |
|                                  | modélisation mathématique. Ce    |
|                                  | projet se décline dans 6 pays    |
|                                  | hébergeant un institut du        |
|                                  | Pasteur Network (PN), avec des   |
|                                  | activités adaptées selon les     |
|                                  | situations épidémiologiques au   |
|                                  | sein de chaque pays.             |
|                                  |                                  |
|                                  | Le protocole VAXPOX-CI a pour    |
|                                  | objectif de présenter les        |
|                                  | activités de recherche qui       |
|                                  | seront conduites en Côte         |
|                                  | d’Ivoire par l’Institut Pasteur  |
|                                  | de Côte d’Ivoire (IPCI),         |
|                                  | laboratoire national de          |
|                                  | référence (LNR) pour le          |
|                                  | diagnostic mpox. L’ensemble des  |
|                                  | WP seront déclinés dans ce pays  |
|                                  | qui rapporte de plus en plus de  |
|                                  | cas de mpox depuis 2024.         |
+----------------------------------+----------------------------------+
| 7.  Objectif principal           | -   Estimer la séroprévalence    |
|                                  |       des anticorps anti-MPXV    |
|                                  |       par âge, sexe et groupe à  |
|                                  |       risque dans différentes    |
|                                  |       populations urbaines et    |
|                                  |       rurales en Côte d’Ivoire   |
+----------------------------------+----------------------------------+
| 8.  Objectifs secondaires        | -   Identifier les facteurs      |
|                                  |       associés à l’infection par |
|                                  |       le MPXV dans différentes   |
|                                  |       populations urbaines et    |
|                                  |       rurales en Côte d’Ivoire   |
|                                  |                                  |
|                                  | -   Estimer la prévalence de     |
|                                  |       l’intention vaccinale mpox |
|                                  |       dans différentes           |
|                                  |       populations urbaines et    |
|                                  |       rurales en Côte d’Ivoire   |
|                                  |                                  |
|                                  | -   Estimer la séroprévalence    |
|                                  |       des anticorps dirigés      |
|                                  |       contre un panel de         |
|                                  |       maladies infectieuses      |
|                                  |       d’intérêt en santé         |
|                                  |       publique                   |
|                                  |                                  |
|                                  | -   Quantifier les contributions |
|                                  |       respectives de la          |
|                                  |       transmission zoonotique et |
|                                  |       de la transmission         |
|                                  |       interhumaine du MPXV dans  |
|                                  |       la dissémination du mpox   |
|                                  |       en Côte d’Ivoire           |
|                                  |                                  |
|                                  | -   Modéliser la circulation du  |
|                                  |       mpox en Côte d’Ivoire sur  |
|                                  |       la base des données        |
|                                  |       collectées                 |
|                                  |                                  |
|                                  | -   Identifier les stratégies    |
|                                  |       vaccinales les plus        |
|                                  |       efficientes dans le        |
|                                  |       contexte de la Côte        |
|                                  |       d’Ivoire                   |
+----------------------------------+----------------------------------+
| 9.  Critère d’évaluation         | -   Séroprévalence des anticorps |
|     principal                    |       IgM et IgG anti-MPXV par   |
|                                  |       âge et sexe dans           |
|                                  |       différentes populations    |
|                                  |       urbaines et rurales en     |
|                                  |       Côte d’Ivoire              |
+----------------------------------+----------------------------------+
| 10. Critères d’évaluation        | -   Facteur d’augmentation (ou   |
|     secondaires                  |       de diminution) du risque   |
|                                  |       d’infection passée par le  |
|                                  |       MPXV selon les             |
|                                  |       caractéristiques des       |
|                                  |       sujets participant à       |
|                                  |       l’étude                    |
|                                  |                                  |
|                                  | -   Prévalence de l’intention    |
|                                  |       vaccinale mpox dans        |
|                                  |       différentes populations    |
|                                  |       urbaines et rurales en     |
|                                  |       Côte d’Ivoire              |
|                                  |                                  |
|                                  | -   Séroprévalence des anticorps |
|                                  |       dirigés contre un panel de |
|                                  |       maladies infectieuses      |
|                                  |       d’intérêt en santé         |
|                                  |       publique                   |
|                                  |                                  |
|                                  | -   Part relative des            |
|                                  |       introductions zoonotiques  |
|                                  |       versus cas de transmission |
|                                  |       interhumaine en Côte       |
|                                  |       d’Ivoire                   |
|                                  |                                  |
|                                  | -   Modèle mathématique          |
|                                  |       décrivant la circulation   |
|                                  |       du mpox en Côte d’Ivoire,  |
|                                  |       calibré aux données        |
|                                  |       collectées dans le projet  |
|                                  |                                  |
|                                  | -   Résultats d’analyses         |
|                                  |       coût-efficacité de         |
|                                  |       différentes stratégies     |
|                                  |       vaccinales en Côte         |
|                                  |       d’Ivoire                   |
+----------------------------------+----------------------------------+
| 11. Echantillons biologiques     | -   Un prélèvement de sang       |
|                                  |       réalisé chez les           |
|                                  |       participants au cours des  |
|                                  |       enquêtes transversales de  |
|                                  |       séroprévalence (groupe     |
|                                  |       représentatif de la        |
|                                  |       population générale en     |
|                                  |       zone rurale, groupes à     |
|                                  |       risque d’infection par     |
|                                  |       voie sexuelle en zone      |
|                                  |       urbaine/périurbaine) : 4   |
|                                  |       ml de sang veineux chez    |
|                                  |       les participants de 12 ans |
|                                  |       et plus, 2 ml de sang      |
|                                  |       veineux chez les           |
|                                  |       participants 6 à 11 ans,   |
|                                  |       ou environ 250 µl de sang  |
|                                  |       capillaire chez les        |
|                                  |       participants de 1 à 5 ans. |
|                                  |                                  |
|                                  | -   Sérums issus de la biobanque |
|                                  |       du Centre National de      |
|                                  |       Transfusion Sanguine de    |
|                                  |       Côte d’Ivoire (CNTSCI),    |
|                                  |       collectés auprès de        |
|                                  |       donneurs de sang en zone   |
|                                  |       urbaine/périurbaine.       |
|                                  |                                  |
|                                  | -   Ecouvillons de pus ou croûte |
|                                  |       issus de la surveillance   |
|                                  |       des cas suspects de mpox   |
|                                  |       (cas passés et cas qui     |
|                                  |       seraient détectés pendant  |
|                                  |       la durée de l’étude).      |
+----------------------------------+----------------------------------+
| 12. Méthode - déroulement de la  | Le projet VAXPOX-CI s’articule   |
|     recherche                    | autour de 4 Work Packages (WP) : |
|                                  |                                  |
|                                  | WP1 Epidémiologie et WP2         |
|                                  | Sérologie : réalisation          |
|                                  | d’enquêtes transversales de      |
|                                  | séroprévalence auprès de         |
|                                  | différentes populations (la      |
|                                  | détection des anticorps          |
|                                  | spécifiques anti-MPXV dans les   |
|                                  | échantillons de sérum témoignera |
|                                  | d’une infection passée), pour    |
|                                  | déterminer les populations les   |
|                                  | plus affectées par le mpox :     |
|                                  |                                  |
|                                  | -   Enquêtes comprenant un       |
|                                  |       questionnaire standardisé  |
|                                  |       et un prélèvement sanguin, |
|                                  |       effectuées auprès de       |
|                                  |       personnes appartenant à    |
|                                  |       des groupes                |
|                                  |       particulièrement à risque  |
|                                  |       de transmission du MPXV    |
|                                  |       par voie sexuelle en zone  |
|                                  |       urbaine/périurbaine, et    |
|                                  |       auprès de personnes        |
|                                  |       représentatives de la      |
|                                  |       population générale en     |
|                                  |       zone rurale.               |
|                                  |                                  |
|                                  | -   Etude sur des échantillons   |
|                                  |       issus de la biobanque du   |
|                                  |       CNTSCI, collectés auprès   |
|                                  |       de donneurs de sang en     |
|                                  |       zone urbaine/périurbaine   |
|                                  |       (groupe représentatif de   |
|                                  |       la population générale     |
|                                  |       adulte).                   |
|                                  |                                  |
|                                  | WP3 Virologie moléculaire :      |
|                                  | séquençage des souches virales   |
|                                  | isolées des cas confirmés de     |
|                                  | mpox, à partir des échantillons  |
|                                  | issus de la surveillance         |
|                                  | nationale en Côte d’Ivoire, pour |
|                                  | déterminer le caractère          |
|                                  | zoonotique ou interhumain de la  |
|                                  | transmission du MPXV.            |
|                                  |                                  |
|                                  | WP4 Modélisation mathématique :  |
|                                  | élaboration de modèles           |
|                                  | mathématiques décrivant la       |
|                                  | circulation du mpox en Côte      |
|                                  | d’Ivoire, calibrés grâce aux     |
|                                  | données collectées dans les      |
|                                  | autres WP ; réalisation          |
|                                  | d’analyses coût-efficacité de    |
|                                  | stratégies vaccinales pour       |
|                                  | identifier les plus efficientes  |
|                                  | dans le contexte de la Côte      |
|                                  | d’Ivoire.                        |
+----------------------------------+----------------------------------+
| 13. Populations étudiées         | -   Sujets appartenant à des     |
|                                  |       groupes à risque           |
|                                  |       d’infections par voie      |
|                                  |       sexuelle en zone           |
|                                  |       urbaine/périurbaine        |
|                                  |       (travailleurs du sexe      |
|                                  |       (TS), hommes ayant des     |
|                                  |       relations sexuelles avec   |
|                                  |       des hommes (HSH), patients |
|                                  |       référés pour diagnostic    |
|                                  |       biologique d’infections    |
|                                  |       sexuellement               |
|                                  |       transmissibles (IST))      |
|                                  |                                  |

Justification scientifique DE LA RECHERCHE

Contexte. La variole du singe, à présent appelée mpox, a été rapportée
chez l'homme pour la première fois en 1970. Jusqu’en 2022, le virus
causant cette maladie (MPXV) ne provoquait que des épidémies limitées
d’origine zoonotique après contact avec des animaux sauvages, notamment
des écureuils (Funisciurus) et des rats (Cricetomys), et une
transmission interhumaine restreinte, principalement par des contacts
étroits avec les fluides biologiques et les lésions cutanées des
personnes infectées¹. Deux clades génétiques du MPXV ont été
initialement décrits : le clade I en Afrique centrale et le clade II en
Afrique de l’Ouest². Jusqu’en 2017, le clade I prédominait et était
connu pour causer une maladie plus sévère que le clade II, avec un taux
de létalité (CFR) estimé à 10,6 %. Le clade II était considéré comme
moins sévère, avec un CFR de 3,6 %³. Le CFR variait selon l’âge : il
était estimé à 9,7 % chez les enfants de 5 ans et à 1,2 % chez les
sujets de 30 ans pour le clade I⁴. Après quatre décennies sans nouveaux
cas rapportés, le MPXV clade II a réémergé au Nigeria en 2017⁵, suivi
d’une pandémie en 2022 touchant des pays non endémiques, avec des
transmissions interhumaines soutenues d’un sous-clade appelé IIb,
principalement par contacts sexuels entre hommes ayant des relations
sexuelles avec des hommes (HSH^(6,7)). En juillet 2022, l’Organisation
mondiale de la santé (OMS) a déclaré le mpox comme une Urgence de Santé
Publique de Portée Internationale (USPPI). Cette urgence a duré un an,
avec près de 100 000 cas rapportés dans plus de 100 pays et un CFR
inférieur à 0,1 %⁸. Des campagnes de prévention et de vaccination
ciblées auprès des populations à haut risque de transmission par voie
sexuelle, principalement les HSH, ont permis de contenir
l’épidémie^(9,10). Alors que cette épidémie due au clade IIb déclinait,
les cas dus au clade I ont progressivement augmenté en République
Démocratique du Congo (RDC) et le virus s’est propagé à des pays
d’Afrique de l’Est jusque-là épargnés, y compris dans des zones
urbaines. Depuis 2024, la dynamique de l’épidémie en Afrique suggère,
pour les pays nouvellement touchés, l’introduction du virus via les
populations à haut risque de transmission par voie sexuelle, notamment
dans la classe d’âge 15-44 ans, les enfants pouvant être infectés
secondairement^(11,12). D’août 2024 à septembre 2025, l’OMS a de nouveau
déclaré le mpox comme USPPI¹³, en raison de l'apparition d’un nouveau
sous-clade dénommé clade Ib, de la transmission interhumaine soutenue
observée (y compris par voie sexuelle¹⁴) et de la diffusion du MPXV de
régions historiquement rurales et forestières vers de nouvelles zones, y
compris urbaines¹⁵. Depuis, plusieurs pays africains ont rapporté une
transmission communautaire soutenue du clade Ib, et 14 pays en-dehors du
continent africain ont rapporté des cas sporadiques liés à des retours
de voyage^(16,17). La levée de l’alerte USPPI en septembre 2025 ne
marque pas la fin de la lutte contre le mpox, avec de nombreuses
épidémies toujours en cours sur le continent africain.

Vaccins disponibles. La réponse à cette épidémie doit être adaptée aux
caractéristiques épidémiologiques et cliniques spécifiques des
différentes épidémies simultanées sur le continent africain¹⁸. Dans ce
contexte, il est important d’examiner le rôle des vaccins parmi les
outils de prévention¹⁹, particulièrement après les résultats décevants
d’un essai contrôlé randomisé de phase III portant sur le seul
médicament disponible, le Tecovirimat, en RDC²⁰. Le vaccin utilisé lors
de la pandémie de 2022, Modified Vaccinia Ankara (MVA), est une forme
atténuée du vaccin antivariolique fabriqué par Bavarian Nordic. Son
efficacité contre le MPXV clade IIb a été estimée à 76 % après une dose
et 82 % après deux doses²¹. Bien que son efficacité contre d’autres
clades du MPXV ne soit pas encore connue, une protection croisée est
attendue, comme cela a été observé avec les vaccins antivarioliques de
première génération, qui conféraient une protection de 85 % contre le
mpox²². Le vaccin MVA a reçu une préqualification de l’OMS pour une
utilisation uniquement chez les enfants de plus de 12 ans. Un autre
vaccin, appelé LC16m8, développé il y a plus de 20 ans au Japon, a reçu
le 19 novembre 2024 une autorisation d’utilisation d’urgence de l’OMS
pour les adultes et les enfants de plus d’un an, en dose unique²³, avec
plus de 3 millions de doses offertes par le gouvernement japonais. En
1973-1974, ce vaccin a été administré à environ 50 000 enfants japonais
âgés de 1 à 7 ans, avec quelques effets secondaires cutanés et des
épisodes d’auto-inoculation²⁴. Des questions de responsabilité juridique
pourraient retarder la livraison des doses aux pays concernés, et son
administration avec des aiguilles bifurquées pourrait réduire son
acceptabilité. En janvier 2025, l’Autorité de Régulation Pharmaceutique
de la RDC a autorisé l’inclusion de tous les enfants à partir d’1 an
dans les campagnes de vaccination. Enfin, en date de février 2025, six
millions de doses ont été promises au continent africain¹⁷.

Stratégies vaccinales. Bien que ces vaccins élargissent l’arsenal pour
freiner la propagation du mpox, les stratégies à mettre en place restent
incertaines. Étant donné les stocks limités de vaccins, une vaccination
généralisée n'est pas envisageable. La stratégie recommandée par le CDC
Africa, l’OMS et d’autres organisations est de cibler les contacts de
cas confirmés, les professionnels de santé, les autres acteurs de
première ligne, et les populations clé à risque d’infection dans des
zones avec transmission interhumaine active²⁵. Il est urgent de disposer
de données scientifiques afin d’éclairer au mieux ces stratégies et
d’identifier les populations clés à cibler par les campagnes de
vaccination. Les connaissances actuelles sur la transmission du MPXV
identifient les groupes potentiels suivants à cibler pour vaccination :
les professionnels de santé exposés aux patients ; en zone rurale, les
enfants en contact avec les réservoirs animaux (souvent via le piégeage
d’animaux sauvages ou d’autres activités) qui sont plus à risque de
formes sévères de la maladie ; en zone urbaine, les populations à risque
élevé de transmission par voie sexuelle (TS, HSH, patients atteints
d’IST, chauffeurs routiers). Cependant, il n’est pas précisément établi
quel(s) groupe(s) à risque devraient être priorisés pour les campagnes
de vaccination pour un impact maximal dans un contexte de limitation de
la quantité de vaccins. Une étude a recommandé la vaccination des
enfants dans les régions endémiques de la RDC²⁶, mais la migration
urbaine récente du virus et sa transmission soutenue dans les réseaux
sexuels²⁷ n’ont pas été pris en compte. À la suite de cette publication,
la nécessité d’analyses coût-efficacité de différentes stratégies
vaccinales a été mise en avant par plusieurs experts^(28–31). Néanmoins,
cette évaluation essentielle pour orienter les décisions de santé
publique semble impossible en l’absence de données sur le fardeau réel
de l’infection dans les groupes à risque, et les contributions
respectives des différentes voies de transmission. Aussi, la littérature
indique que l’acceptabilité vaccinale mpox en Afrique est parmi les plus
faibles au monde^(32,33), mais des données nationales ne sont pas
disponibles pour tous les pays. Mesurer l’intention de vaccination parmi
les groupes à risque d’infection est essentiel pour favoriser une
couverture vaccinale optimale.

Situation épidémiologique en Côte d’Ivoire. En Côte d’Ivoire, l’Institut
Pasteur de Côte d’Ivoire (IPCI) est le Laboratoire National de Référence
(LNR) pour le mpox. Jusqu’à 2024, seulement deux cas humains de mpox
avaient été rapportés dans le pays, en 1971 et 1981³⁴. En 2012, un cas
de mpox fatal a été confirmé chez un chimpanzé³⁵. Depuis la confirmation
d’un nouveau cas humain en Côte d’Ivoire en juillet 2024, au 25 août
2025, 793 cas suspects ont été déclarés, dont 174 confirmés en
laboratoire. Le virus circule à la fois en zones rurales et en zones
urbaines, affectant près d’un tiers des districts sanitaires en Côte
d’Ivoire, principalement dans les régions ouest et est du pays. Lorsque
le séquençage a été réalisé, les clades IIa et IIb ont été identifiés.
On ne peut pas écarter cependant que la circulation du virus en Côte
d’Ivoire soit encore plus importante et non détectée, soit du fait du
caractère reculé des zones rurales où les transmissions d’origine
zoonotique ont lieu, soit du fait de formes cliniques
pauci-symptomatiques, notamment lors des transmissions par voie sexuelle
en milieu urbain. Des enquêtes sérologiques permettant l’identification
d’infections passées et parfois peu symptomatiques seraient un
complément important de la surveillance de la circulation du MPXV en
Côte d’Ivoire.

Objectif du projet. Le projet VAXPOX a pour objectif d’accompagner les
politiques de santé publique à mesure que les vaccins deviennent
disponibles, via une approche pluridisciplinaire associant épidémiologie
(WP1), sérologie (WP2), virologie moléculaire (WP3) et modélisation
mathématique comprenant une analyse coût-efficacité (WP4). Ce projet se
décline dans 6 pays hébergeant un institut du Pasteur Network (PN), avec
des activités adaptées selon les situations épidémiologiques au sein de
chaque pays. Le protocole VAXPOX-CI a pour objectif de présenter les
activités de recherche qui seront conduites en Côte d’Ivoire par l’IPCI.
L’ensemble des WP sera décliné dans ce pays qui connaît une nette
augmentation des cas depuis 2024.

En parallèle de ces objectifs spécifiques au mpox, le projet VAXPOX-CI
sera l’occasion d’évaluer la circulation d’autres pathogènes. En effet,
de nombreux pathogènes d’intérêt pour la santé des populations circulent
souvent dans les mêmes zones des pays à revenu faible ou intermédiaire.
Les nouveaux outils de recherche que sont les tests sérologiques
multiplex permettent une description détaillée de la cinétique des
anticorps spécifiques de plusieurs pathogènes à partir d’un seul
échantillon biologique. Le projet VAXPOX-CI aura donc également pour
objectif d’évaluer la circulation de plusieurs agents infectieux
d’intérêt parmi les populations ciblées dans cette étude.

IMPACTS ATTENDUS DU PROJET

Impacts pour la communauté scientifique

Les résultats des analyses effectuées au sein de chacun des WP du projet
VAXPOX-CI fourniront des données pertinentes pour une meilleure
compréhension de l’épidémiologie du mpox. L’obtention de données de
séroprévalence en population générale et dans des groupes
particulièrement à risque de transmission du virus par voie sexuelle
(WP1-WP2) permettra de quantifier la circulation d’un virus
historiquement restreint à des zones reculées forestières mais ayant à
présent atteint de nouvelles zones géographiques, y compris des zones
urbaines et péri-urbaines qui maximisent la transmission interhumaine.
Grâce à la technologie multiplex, le projet VAXPOX-CI permettra
également d’évaluer la circulation de plusieurs agents infectieux
d’intérêt pour la santé des populations, au sein des groupes ciblés par
cette étude.

Les analyses phylogénétiques des souches virales circulant en Côte
d’Ivoire et dans les pays voisins (WP3) permettront une meilleure
compréhension de l’évolution du MPXV, ainsi que la détection des
nouveaux sous-clades circulant actuellement sur le continent africain et
de sa dissémination dans la région. Enfin, le modèle mathématique
développé pour décrire la circulation du mpox en Côte d’Ivoire, prenant
notamment en compte les différents modes de transmission (WP4) pourra
être adapté à d’autres contextes pour mieux comprendre la dissémination
de ce virus.

Impacts communautaires

Le contrôle de l’épidémie actuelle de mpox est d’une importance capitale
pour les populations, en particulier pour les enfants qui sont les plus
à risque de formes graves de la maladie, mais aussi pour prévenir
l’établissement du mpox en zone urbaine via une transmission
interhumaine soutenue. La réalisation du projet VAXPOX-CI contribuera à
mieux identifier les groupes particulièrement à risque d’infection par
le MPXV, et contribuera à cibler les campagnes de vaccination afin
qu’elles soient les plus efficientes pour limiter la dissémination du
virus. De plus, la communication des résultats du projet (cf. Section 9.
Diffusion des résultats) permettra de sensibiliser les populations à la
contribution relative de la transmission zoonotique et la transmission
interhumaine du MPXV, ainsi qu’au recours à la vaccination pour se
protéger de la maladie. Aussi, l’utilisation de la technologie multiplex
pour évaluer la circulation de plusieurs agents pathogènes présente un
avantage substantiel pour les populations ciblées, car la fourniture
d'un seul échantillon sanguin dans le cadre du projet VAXPOX-CI
contribuera à la surveillance d'un éventail de maladies infectieuses,
sans nécessiter la mise en place de multiples études.

Impacts en santé publique

Les résultats du projet VAXPOX-CI seront particulièrement pertinents
pour les autorités sanitaires de Côte d’Ivoire dans le contexte de
l’épidémie en cours, en documentant la progression de l’épidémie de mpox
au sein des populations ciblées par cette étude, et en identifiant des
groupes particulièrement à risque d’infection. Grâce à la modélisation
et à l’analyse coût-efficacité des stratégies vaccinales, il sera
possible de conseiller les autorités sanitaires sur les groupes à
vacciner en priorité afin de maximiser l’impact des campagnes. Le projet
VAXPOX-CI aura également un impact à long terme car les recommandations
pourront être adaptées à de nouvelles informations (disponibilité des
vaccins, acceptabilité vaccinale, données actualisées de
séroprévalence…) en mettant à jour les modèles mathématiques développés.
Enfin, les résultats de sérologie multiplex contre un panel d’agents
infectieux d’intérêt en santé publique éclaireront les autorités
sanitaires en Côte d’Ivoire sur la prévalence de l'exposition à
plusieurs maladies infectieuses courantes parmi les populations ciblées
par cette étude.

Impacts sur la préparation aux pandémies

Le projet VAXPOX-CI permettra de renforcer les compétences de l’IPCI
concernant des méthodes et des technologies cruciales pour la réponse
aux menaces infectieuses à potentiel pandémique. Premièrement, la
réalisation d’enquêtes sérologiques de terrain avec une méthodologie
hautement standardisée et adaptable à d’autres pathogènes émergents
(ex : technologie Luminex) est essentielle à la compréhension de la
diffusion d’un pathogène au sein de groupes à risque et en population
générale. Ces enquêtes peuvent avoir plusieurs objectifs :
identification des populations à risque, mise en évidence de formes
cliniques asymptomatiques, évaluation de l’immunité acquise après
infection/vaccination, et étude des chaînes de transmission autour des
cas confirmés. Deuxièmement, le séquençage systématique du génome
complet, le partage des séquences d’agents pathogènes au niveau régional
et les analyses phylogénétiques groupées de ces séquences permettent
d’assurer une surveillance quasi en temps réel de la circulation d’un
agent pathogène donné et de sa capacité d’échappement immunitaire,
améliorant ainsi la coordination régionale pour la surveillance grâce à
une couverture et un reporting accrus. Troisièmement, le renforcement
des compétences en modélisation mathématique des épidémies est un
complément essentiel à une meilleure compréhension des risques
épidémiques et des réponses à apporter aux menaces infectieuses. Enfin,
la collaboration entre chercheurs de différents instituts du PN sera
source de bénéfices multilatéraux. Les collaborations nouées dans le
cadre de ce projet de recherche seront déterminantes pour faciliter les
interactions en période de pandémie due à un autre pathogène, permettant
un partage d’expertise au bénéfice de l’ensemble du réseau.

Objectifs

Objectif principal

L’objectif principal de ce projet est d’estimer la séroprévalence des
anticorps anti-MPXV par âge, sexe et groupe à risque dans différentes
populations urbaines et rurales en Côte d’Ivoire.

Objectifs secondaires

Les objectifs secondaires de ce projet sont définis ci-dessous :

-   Identifier les facteurs associés à l’infection par le MPXV dans
      différentes populations urbaines et rurales en Côte d’Ivoire

-   Estimer la prévalence de l’intention vaccinale mpox dans différentes
      populations urbaines et rurales en Côte d’Ivoire

-   Estimer la séroprévalence des anticorps dirigés contre un panel de
      maladies infectieuses d’intérêt en santé publique

-   Quantifier les contributions respectives de la transmission
      zoonotique et de la transmission interhumaine du MPXV dans la
      dissémination du mpox en Côte d’Ivoire

-   Modéliser la circulation du mpox en Côte d’Ivoire sur la base des
      données collectées

-   Identifier les stratégies vaccinales les plus efficientes dans le
      contexte de la Côte d’Ivoire

METHODOLOGIE

Critères d’évaluation

Critère d’évaluation principal

Séroprévalence des anticorps IgM et IgG anti-MPXV par âge et sexe dans
différentes populations urbaines et rurales en Côte d’Ivoire.

Critères d’évaluation secondaires

-   Facteur d’augmentation (ou de diminution) du risque d’infection
      passée par le MPXV selon les caractéristiques des sujets
      participant à la recherche

-   Prévalence de l’intention vaccinale mpox dans différentes
      populations urbaines et rurales en Côte d’Ivoire

-   Séroprévalence des anticorps dirigés contre un panel de maladies
      infectieuses d’intérêt en santé publique

-   Part relative des introductions zoonotiques versus cas de
      transmission interhumaine en Côte d’Ivoire

-   Modèle mathématique décrivant la circulation du mpox en Côte
      d’Ivoire, calibré aux données collectées dans le projet

-   Résultats d’analyses coût-efficacité de différentes stratégies
      vaccinales en Côte d’Ivoire

Plan expérimental

WP1-2 : Epidémiologie et sérologie

Objectif principal

Pour estimer la séroprévalence des anticorps anti-MPXV par âge, sexe et
groupe à risque dans différentes populations urbaines et rurales en Côte
d’Ivoire, des enquêtes de séroprévalence seront menées auprès des
populations suivantes :

-   Sujets représentatifs de la population générale adulte en zone
    urbaine/périurbaine,

-   Sujets représentatifs de la population générale en zone rurale,

-   Sujets issus de groupes à risque de transmission du MPXV par voie
    sexuelle en zone urbaine/périurbaine (TS, HSH et patients référés
    pour diagnostic biologique d’IST).

Pour les sujets représentatifs de la population générale adulte en zone
urbaine/périurbaine, des échantillons seront sélectionnés dans la
biobanque du CNTSCI parmi les donneurs de sang résidant dans le district
d’Abidjan.

Pour les autres populations d’intérêt, des sujets seront recrutés au
cours d’enquêtes au sein de la population générale en zone rurale, et
auprès de groupes particulièrement à risque de transmission du MPXV par
voie sexuelle. Ces enquêtes impliqueront :

-   un questionnaire individuel standardisé pour recueillir des données
    épidémiologiques et cliniques,

-   et un prélèvement de sang (4 ml de sang veineux pour les
    participants âgés de 12 ans et plus, 2 ml pour les enfants âgés de 6
    à 11 ans et environ 250 µl de sang capillaire pour enfants âgés de 1
    à 5 ans).

Les analyses sérologiques des échantillons détecteront les éventuelles
expositions passées au MPXV grâce à un test immunologique multiplex sur
microsphères (MMIA) basé sur la technologie Luminex, détectant
simultanément les anticorps IgM et IgG dirigés contre plus de 35
antigènes du MPXV. Ce test a été développé à l’IPP et sera transféré aux
équipes de l’IPCI, formées à l’utilisation de la technologie Luminex.

Les données sérologiques mpox obtenues et les données épidémiologiques
associées seront partagées dans une base de données commune avec les
autres instituts du PN déclinant le projet VAXPOX, permettant d’élargir
la surveillance collaborative entre les instituts du réseau à un volet
sérologique. 

Objectifs secondaires

Afin de répondre à plusieurs objectifs secondaires de cette recherche,
les éléments suivants seront mis en œuvre :

-   Identifier les facteurs associés à l’infection par le MPXV selon les
      groupes à risque en Côte d’Ivoire : Dans l’éventualité où des
      sous-groupes parmi les populations ciblées seraient séropositifs
      pour le mpox, les données collectées dans le questionnaire
      administré aux participants permettront d’évaluer les facteurs
      d’augmentation (ou de diminution) du risque d’infection passée par
      le MPXV selon les caractéristiques des participants.

-   Estimer la prévalence de l’intention vaccinale mpox dans différentes
      populations urbaines et rurales en Côte d’Ivoire : Le
      questionnaire administré aux participants comportera des questions
      se rapportant à l’acceptabilité du vaccin mpox, ce qui permettra
      d’évaluer la prévalence de l’intention vaccinale dans les
      populations ciblées.

-   Estimer la séroprévalence des anticorps dirigés contre un panel de
      maladies infectieuses d’intérêt en santé publique : La technologie
      multiplex Luminex utilisée pour répondre à l’objectif principal
      repose sur l’utilisation de billes enrobées d’antigènes,
      permettant de mesurer avec précision un grand nombre d’anticorps à
      partir d’un petit volume de sérum ou de plasma. Le test utilisé
      pour le projet permettra d’évaluer la prévalence des anticorps
      spécifiques contre un panel d’agents pathogènes d’intérêt, dans
      les populations ciblées.

WP3 : Virologie moléculaire

Pour répondre à l’objectif secondaire « quantifier les contributions
respectives de la transmission zoonotique et de la transmission
interhumaine du MPXV » dans la dissémination du mpox en Côte d’Ivoire,
des analyses génomiques seront menées à partir d’échantillons
(pus/croûtes de lésions cutanées) et leurs données associées, issus de
la surveillance nationale à laquelle l’IPCI prend part en tant que LNR
pour le diagnostic mpox. Les analyses conduites dans le cadre de la
recherche incluront le séquençage complet des souches de MPXV de cas
confirmés de mpox et des analyses phylogénétiques.

1.  Les échantillons provenant de cas confirmés détectés par le système
      de surveillance pendant la durée de la recherche et présentant un
      seuil de cycle PCR MPXV inférieur à 30 seront sélectionnés en
      priorité pour séquençage à l’IPCI afin d’obtenir des génomes
      complets. Parmi les 123 échantillons issus de cas confirmés passés
      en biobanque à l’IPCI, ceux présentant des charges virales
      suffisantes feront également l’objet d’un séquençage complet. Pour
      cela, un dosage préalable au Qubit 5 ou au Nanodrop One C sera
      réalisé sur les échantillons sélectionnés pour évaluer leur
      concentration (objectif : 10-100 ng/ µL) et leur pureté
      (objectif : 1,8-2). Les librairies pour séquençage seront
      préparées selon les conditions du fabricant avec le Rapid
      Barcoding kit (SQK-RBK110) et la technologie Oxford Nanopore
      Technology (ONT) sera utilisée pour le séquençage.

2.  Les données de séquençage seront utilisées pour déterminer les
      clades, et la présence ou l’absence de signatures moléculaires de
      transmission interhumaine (ex : celles laissées par l’enzyme
      humaine APOBEC3) sera recherchée, afin de déterminer la part des
      infections d’origine zoonotique et celle des infections
      interhumaines parmi les infections en Côte d’Ivoire.

3.  Les séquences virales obtenues et leurs données épidémiologiques
      associées seront partagées dans une base de données commune avec
      les autres instituts du PN déclinant le projet VAXPOX, permettant
      de renforcer la surveillance génomique collaborative entre les
      instituts du réseau. Un modèle phylodynamique sera développé en
      utilisant l’ensemble des séquences disponibles dans les banques de
      données publiques, afin de décrire la dynamique de transmission
      zoonotique et interhumaine.

WP4 : Modélisation mathématique

Pour répondre aux objectifs secondaires « modéliser la circulation du
mpox en Côte d’Ivoire sur la base des données collectées » et
« identifier les stratégies vaccinales les plus efficientes dans le
contexte de la Côte d’Ivoire », les données collectées dans les autres
WP seront utilisées et complétées avec les données issues de la
surveillance nationale en Côte d’Ivoire. Des modèles mathématiques
décrivant la circulation du mpox en Côte d’Ivoire seront développés et
serviront à évaluer différentes stratégies de vaccination pour
identifier les stratégies les plus efficientes.

Les étapes qui seront entreprises dans ce WP sont les suivantes :

1.  Un modèle mathématique décrivant la propagation du mpox sera généré
      en prenant en compte les différentes voies de transmission du
      MPXV, y compris la transmission interhumaine sexuelle ou non
      sexuelle, ainsi que la transmission zoonotique. Le déclin attendu
      des titres d’anticorps à distance de l’infection sera pris en
      compte dans l’analyse des résultats des enquêtes sérologiques. La
      population modélisée sera structurée par âge et par groupe à
      risque.

2.  Un échantillonnage par chaîne de Markov Monte Carlo sera utilisé
      pour calibrer le modèle aux données concernant les cas confirmés
      de mpox en Côte d’Ivoire collectées par le système national de
      surveillance. Cette calibration permettra de quantifier la
      contribution des différentes voies de transmission à la
      propagation globale.

3.  Le modèle obtenu servira à simuler des épidémies selon différentes
      stratégies de vaccination. La charge infectieuse évitée par chaque
      stratégie sera estimée, et celle ayant l’impact maximal sera
      identifiée. Le modèle sera calibré aux données de Côte d’Ivoire
      provenant de contextes variés (urbain vs. rural) afin de
      déterminer les spécificités propres à chaque contexte, et
      d’évaluer si ces différences justifient l’application de
      stratégies vaccinales différenciées.

4.  Des analyses coût-efficacité des stratégies de vaccination
      envisageables en Côte d’Ivoire seront réalisées, en prenant en
      compte les différences de gravité de la maladie selon certaines
      caractéristiques individuelles, telles que l’âge ou le clade
      viral. Le fardeau de la maladie évité estimé par le modèle de
      transmission sera converti en mesures économiques ou sanitaires.
      Cette harmonisation des résultats du modèle permettra de calculer
      les ratios coût-efficacité incrémentaux et d’offrir aux décideurs
      des informations sur la valeur des différentes stratégies
      vaccinales. Des analyses de sensibilité seront menées pour évaluer
      la transférabilité des résultats à d'autres contextes
      épidémiologiques.

Population étudiée

Description des populations étudiées

-   Population générale en zone rurale : personnes résidant dans les
      ménages sélectionnés en zone rurale dans le département de Dabou
      (50 km à l’ouest d’Abidjan).

-   Population générale en zone urbaine/périurbaine : échantillons du
      CNTSCI provenant de donneurs de sang résidant dans le district
      d’Abidjan.

-   Groupes à risque de transmission du virus par voie sexuelle :

-   TS consultant dans les centres de prise en charge de l’ONG Espace
    Confiance à Abidjan,

-   HSH consultant dans les centres de prise en charge de l’ONG Espace
    Confiance à Abidjan,

-   Patients référés pour diagnostic biologique d’IST au CNR des IST à
    l’IPCI à Abidjan.

Ces cinq groupes ciblés par les enquêtes de séroprévalence sont
complémentaires en termes d’expositions potentielles au mpox (exposition
zoonotique pour la population générale dans le département de Dabou,
expositions en zone urbaine pour les donneurs de sang dans le district
d’Abidjan, exposition sexuelle pour les TS, HSH et individus porteurs
d’IST). Le projet VAXPOX-CI permettra d’estimer la circulation passée du
MPXV parmi ces groupes, et d’évaluer leur acceptabilité vis-à-vis du
vaccin mpox qui pourrait leur être proposé dans le cadre de campagnes de
vaccination.

-   Cas confirmés de mpox : échantillons provenant des cas confirmés
      détectés dans le cadre du système de surveillance nationale.
      L’analyse des séquences virales isolées des cas confirmés de mpox
      permettra notamment d’évaluer les parts respectives de la
      transmission zoonotique et interhumaine parmi les cas confirmés en
      Côte d’Ivoire.

Justification du nombre de sujets à inclure dans la recherche

WP1-2 Epidémiologie et sérologie

Objectif principal

Pour estimer la séroprévalence des anticorps anti-MPXV par âge, sexe et
groupe à risque dans différentes populations urbaines et rurales en Côte
d’Ivoire, 1000 sujets seront recrutés au sein de populations à risque de
transmission du virus par voie sexuelle, et parmi la population générale
en zone rurale. Ces sujets seront répartis comme suit :

-   200 TS à Abidjan

-   200 HSH à Abidjan

-   200 patients IST à Abidjan

-   400 sujets en zone rurale dans le département de Dabou

300 échantillons complémentaires seront utilisés, collectés par le
CNTSCI auprès de donneurs de sang en zone urbaine/périurbaine, groupe
représentatif de la population générale adulte dans cette zone.

Justification :

-   Pour les groupes particulièrement à risque de transmission du virus
      par voie sexuelle, homogènes en termes de catégorie d’âge, 200
      participants seront recrutés : avec 200 participants, les
      intervalles de confiance exacts binomiaux à 95% pour les
      estimations de séroprévalence de 3%, 10% et 50% seront
      respectivement de (1.1-6.4%), (6.2-15.0%), et (42.9-57.1%).

-   Pour les populations en zones rurales, 400 sujets seront recrutés au
    sein de ménages. En effet, nous doublons la taille de l’échantillon
    pour prendre en compte l’effet du plan de sondage en grappe que nous
    fixons à 2 en l’absence d’estimations du coefficient de corrélation
    intra-grappe. Cela permet d’obtenir la même précision autour des
    estimations de séroprévalence que pour 200 participants.

-   Pour les échantillons du CNTSCI collectés auprès de donneurs de
      sang, 300 échantillons seront utilisés, permettant un gain de
      précision sans recrutement de sujets nécessaire : avec 300
      participants les intervalles de confiance exacts binomiaux à 95%
      pour les estimations de séroprévalence de 3%, 10% et 50% seront
      respectivement de (1.4-5.6%), (6.8-14.0%), et (44.2-55.8%).

WP3 : Virologie moléculaire

Pour répondre à l’objectif secondaire de la recherche « quantifier les
contributions respectives de la transmission zoonotique et de la
transmission interhumaine du MPXV dans la dissémination du mpox en Côte
d’Ivoire », et afin de disposer d’une photographie la plus exhaustive
possible, nous inclurons tous les prélèvements et données de cas
confirmés de mpox reçus à l’IPCI dans le cadre de la surveillance
nationale (cas passés et cas qui surviendraient au cours des 2 ans de la
recherche).

Critères d’inclusion

Populations représentatives de la population générale 

Population générale adulte en zone urbaine et périurbaine :

-   Sérums issus de la biobanque du CNTSCI,

-   collectés auprès de donneurs de 18 ans ou plus résidant dans le
    district d’Abidjan,

-   entre le 1er juillet 2023 et le 31 décembre 2023,

-   ou entre le 1er janvier 2025 et le 30 juin 2025.

Population générale en zone rurale :

-   Personne âgée de 1 an ou plus,

-   résidant en zone rurale dans le département de Dabou,

-   ayant consenti ou dont le responsable légal a consenti à sa
    participation à l’enquête de séroprévalence (questionnaire et
    prélèvement de sang).

Groupes à risque de transmission du virus par voie sexuelle 

TS et HSH :

-   Personne âgée de 18 ans ou plus,

-   consultant au sein d’un centre de prise en charge de l’ONG Espace
    Confiance à Abidjan,

-   ayant consenti à participer à l’enquête de séroprévalence
    (questionnaire et prélèvement de sang veineux).

Patients IST :

-   Personne âgée de 18 ans ou plus,

-   référée au CNR des IST de l’IPCI pour un diagnostic biologique
    d’IST,

-   ayant consenti à participer à l’enquête de séroprévalence
    (questionnaire et prélèvement de sang veineux).

Echantillons d’individus infectés par le MPXV

-   Prélèvements de pus/croûte positifs au MPXV,

-   issus de la surveillance nationale du mpox et pouvant être
    réutilisés à des fins de recherche.

Critères de non-inclusion

Critères de non-inclusion de sujets :

-   Individu dont l'état de santé n'est pas compatible avec un
    prélèvement sanguin selon l'investigateur,

-   Individu sous mesure de protection juridique,

-   Individu hors d’état de consentir,

-   Individu nécessitant une prise en charge médicale immédiate

Critères de non-inclusion d’échantillons :

-   Echantillons de sérums de la biobanque du CNTSCI non exploitables
    (volume insuffisant),

-   Prélèvements de pus/croûte obtenus par l’IPCI dans le cadre de la
    surveillance du mpox, avec une charge virale insuffisante pour
    procéder au séquençage complet du génome viral.

Déroulement pratique de la recherche

Modalités spécifiques aux enquêtes sérologiques (WP1-2)

Les modalités de recrutement des participants pour cette enquête
transversale seront adaptées à chaque contexte, dans le respect des
principes de confidentialité et de non-stigmatisation, notamment pour
les populations exposées à des risques de discrimination. Une attention
particulière sera portée à la formulation des critères d’inclusion, à la
langue utilisée, et à la nature du consentement éclairé.

Les sections ci-dessous ont pour objectif de décrire les différentes
modalités de déroulement pratique de la recherche en fonction des
populations d’intérêt.

Enquête transversale en population rurale dans le département de Dabou

Sélection des villages et des participants

En l’absence d’une base de sondage détaillée et pour des raisons
logistiques, la sélection des villages reposera sur une méthode de
sondage en grappes, similaire à celle utilisée pour les enquêtes de
couverture vaccinale ou enquêtes nutritionnelles. Pour cela, une liste
des villages du département de Dabou, accompagnée d’estimations de
population, sera établie. Parmi ces villages, 30 grappes seront
sélectionnées par tirage au sort, ce qui permettra d’identifier les
localités retenues pour l’étude. La mise en œuvre de l’étude exigera
donc une base de sondage regroupant les villages ainsi que des données
de recensement actualisées afin de procéder au tirage des grappes.

Les participants seront recrutés en suivant un sondage en grappes à deux
niveaux : dans un premier temps, la sélection des villages (premier
niveau) se fera par la méthode des totaux cumulés, en fonction de la
population recensée ; dans un second temps, au sein de chaque village,
les ménages (second niveau) seront choisis de manière aléatoire par la
méthode des itinéraires. Tous les membres des ménages sélectionnés
seront inclus dans l’étude.

Pour chaque grappe, le premier ménage sera désigné par tirage au sort.
Les ménages suivants seront sélectionnés en suivant un itinéraire défini
selon une direction aléatoire (droite ou gauche) à partir du premier
foyer, et ce, jusqu’à atteindre le nombre requis de sujets par grappe,
fixé à 14. Lorsque plusieurs grappes concernent un même village, la
procédure sera répétée pour chacune d’entre elles avec un nouveau tirage
du premier ménage. En cas d’intersection au cours du parcours, la
direction à suivre sera également déterminée par tirage au sort.

Compte tenu d’un effectif moyen de 4,5 personnes dans les ménages, il
convient de sélectionner 89 ménages pour cette étude afin de pouvoir
recruter 400 personnes. Ainsi, trois ménages seront enquêtés dans
chacune des 30 grappes retenues. Afin d’optimiser la variance entre les
unités enquêtées, les ménages adjacents au premier sélectionné seront
évités. Si le nombre requis n’est pas atteint après avoir enquêté les
trois ménages, un foyer supplémentaire sera inclus. Toutefois, si le
seuil est atteint avant la fin, l’enquête se poursuivra tout de même sur
les trois ménages pour garantir l’équilibre.

Information des autorités locales

Avant le lancement de l’enquête, des visites de courtoisie seront
entreprises auprès des autorités préfectorales, sanitaires et des chefs
des villages sélectionnés afin de les informer de la réalisation de
l’étude et de leur présenter le projet et son autorisation de mise en
œuvre. Une lettre d’information ainsi qu’un résumé des objectifs et
procédures de l’étude seront transmis aux autorités concernées,
permettant ainsi de fixer une date pour le déroulement de l’enquête sur
le terrain.

Information des participants

Chaque participant recevra oralement des explications détaillées sur les
objectifs, les modalités, les bénéfices et les risques liés à la
participation, ainsi que sur la gestion confidentielle des données. Il
lui sera clairement indiqué que sa participation est entièrement
volontaire et qu’il peut se retirer de l’étude à tout moment, sans
justification ni conséquence.

Ces informations lui seront délivrées par l’équipe de terrain, au moyen
d’une note d’information, décrivant l’enquête de séroprévalence, remise
avant la proposition du questionnaire et la réalisation du prélèvement
sanguin. Si une difficulté de compréhension survient, l’aide d’un
traducteur sera proposée.

Les analyses réalisées dans le cadre de cette étude sont gratuites pour
le participant qui recevra son résultat de sérologie mpox. Il convient
de souligner que ce résultat correspond à une donnée de recherche et
qu’il sera présenté et expliqué en détail lors de la restitution
individuelle afin d’en garantir la compréhension complète par le
participant. Pour la population résidant dans la zone rurale de Dabou,
la remise du résultat individuel sera assurée sur place par les médecins
de l’IPCI avec le soutien des services sanitaires locaux. Ce déplacement
spécifique a pour objectif d’assurer une restitution claire, adaptée et
compréhensible pour chaque participant, en tenant compte du contexte
local et des besoins spécifiques de cette population.

Consentement éclairé

Le consentement éclairé sera recueilli par le médecin-investigateur
auprès de chaque participant avant toute inclusion dans l’étude. Pour
les mineurs, l’accord des parents ou d’un responsable légal sera
sollicité. Les enfants âgés de 7 à 17 ans recevront une explication,
adaptée à leur âge, et leur assentiment sera également demandé.

Une autorisation sera également requise pour le prélèvement biologique,
l’utilisation des données épidémiologiques et la conservation
d’échantillons pour d’éventuelles recherches ultérieures.

L’inclusion dans l’étude ne sera effective qu’après la lecture du
document d’information en langue locale et la signature du formulaire de
consentement, en double exemplaire. Si le participant ou son
représentant légal ne sait pas lire, un témoin indépendant, extérieur à
l’équipe de recherche, apportera son assistance pour la lecture des
documents. Au besoin, un recours à un traducteur sera effectué pour en
faciliter la compréhension. L’investigateur signera également le
formulaire de consentement.

La note d’information et un exemplaire du formulaire de consentement
seront remis au participant ou à son représentant légal, le cas échéant.
L’investigateur conservera un exemplaire du formulaire de consentement
dans le dossier de l’étude. En complément des documents de l’étude, une
affiche d’information concernant la maladie mpox, ses modes de
transmission et les moyens de s’en protéger, sera transmise à chaque
participant.

Collecte des données et prélèvement biologique

L’équipe de recherche sera constituée d’un médecin, d’un enquêteur et
d’un infirmier qui seront préalablement formés aux procédures de
l’étude. Cette équipe sera accompagnée d’un guide désigné par les
autorités villageoises ou sanitaires.

Un questionnaire standardisé sera administré pour collecter des
informations individuelles (données socio-démographiques et données
médicales) par le biais d’un entretien en face à face. L’infirmier
procédera ensuite au prélèvement de sang (2 à 4 ml de sang veineux pour
les participants de plus de 5 ans, environ 250 µl de sang capillaire
pour les participants de moins de 6 ans). A la suite du prélèvement
sanguin, chaque participant recevra une collation et un kit d’hygiène
sera remis au ménage.

Précisions sur les analyses sérologiques

Les échantillons de sérum issus des enquêtes ou de la biobanque du
CNTSCI seront analysés à l’IPCI avec un test sérologique multiplex à
base de microsphères (MMIA) utilisant la technologie Luminex. Ce test a
été développé à l’IPP (et sera transmis à l’IPCI) pour la détection
simultanée des anticorps IgM ou IgG dirigés contre plus de 35 antigènes
du MPXV et des protéines apparentées des orthopoxvirus (Vaccinia,
Camelpox, Mousepox / virus de l’Ectromélie). La détection des anticorps
dirigés contre d’autres orthopoxvirus permet de contrôler la spécificité
des anticorps détectés pour les antigènes du MPXV. Ce test sérologique
s’est avéré efficace pour détecter des anticorps spécifiques du MPXV
dans des sérums prélevés chez des patients hospitalisés à Paris lors de
l’épidémie de MPXV clade II en 2022, ainsi que chez des cas confirmés de
mpox lors des épidémies de MPXV clade Ia en République Centrafricaine
entre 2021 et 2023 (AUC de 97 % et 96 %, respectivement, données non
publiées). Cette analyse permettra d’établir une estimation de la
séroprévalence des anticorps anti-MPXV par âge et sexe dans les
différentes populations ciblées dans cette recherche.

Au-delà d’antigènes du MPXV, le test comprendra des antigènes permettant
de mesurer des anticorps dirigés contre des agents pathogènes
responsables de maladies infectieuses d’intérêt pour la santé publique,
notamment des maladies évitables par la vaccination (diphtérie, tétanos,
Haemophilus influenza type b, coqueluche, rotavirus, méningite A,
rougeole, rubéole, fièvre jaune), des maladies respiratoires (grippe,
adénovirus, coronavirus), des arboviroses (dengue, Zika, chikungunya),
des parasites du paludisme, les herpes virus, le typhus des
broussailles, le choléra, Borrelia crocidurae, la leptospirose, et des
pathogènes responsables de maladies tropicales négligées (Schistosoma
mansoni, Chlamydia trachomatis, Wucheria bancrofti, Onchocerca volvulus,
Cryptospiridium parvum, Giardia intestinalis, Strongyloides stercoralis,
Entamoeba histolytica). Cette analyse permettra d’établir une
cartographie de la séroprévalence des anticorps pour un panel de
pathogènes d’intérêt dans les populations ciblées dans cette recherche.
Aussi, cela permettra de détecter et corriger les interférences
potentielles entre les anticorps dirigés contre différents pathogènes
présents localement, et les antigènes MPXV utilisés dans le test. Enfin,
ces analyses sérologiques témoigneront de l’exposition passée à
différents pathogènes, elles ne constitueront pas un diagnostic
d’infection en cours. Aucun résultat incident n’est donc à attendre de
ces analyses.

Contrôle des résultats de sérologie

Un sous-groupe d’échantillons pourra être envoyé à l’IPP pour
vérification des résultats par les équipes ayant mis au point le test
sérologique utilisé dans le cadre du projet. L’envoi d’échantillons
serait alors tracé à l’IPCI et à l’IPP. Les échantillons resteront la
propriété de l’IPCI et ne pourront être utilisés ou détruits sans
autorisation de l’IPCI. Les analyses s’effectueront exclusivement par le
personnel de l’IPP autorisé impliqué dans le projet (unités ERI ou
IDEA).

Précisions sur le retour des résultats individuels aux participants

Le résultat individuel de sérologie mpox sera retourné aux participants
via le personnel médical des sites investigateurs (CNR des IST de
l’IPCI, ou centres de consultation de l’ONG Espace Confiance), ou le
personnel médical de l’IPCI accompagné de l’agent de santé local (pour
les ménages du département de Dabou). Etant donnée l’organisation prévue
pour la conduite de la recherche, les échantillons ne seront pas
analysés en temps réel. Si un participant a un résultat positif (IgM+
et/ou IgG+), il lui sera indiqué que le test effectué suggère qu’il y a
des traces dans son sang d’un contact passé avec le virus, et/ou d’une
vaccination contre la mpox. Le texte-type qui accompagnera ce retour
individuel, dans le cas de résultat négatif d’une part, ou de résultat
positif d’autre part, est présenté en annexe 2.

Conservation et réutilisation des échantillons

Les échantillons stockés à l’IPCI et les éventuels échantillons envoyés
à l’IPP restent la propriété de l’IPCI et seront conservés par chaque
institution dans des locaux sécurisés pendant toute la durée du projet +
10 ans (soit 12 ans au total). La conservation des échantillons au-delà
de cette durée devra être autorisée avec l'accord préalable explicite du
Comité National d'Ethique des Sciences de la Vie et de la Santé (CNESVS)
de la Côte d’Ivoire.

Concernant la réutilisation des échantillons pour des recherches
futures, les participants auront la possibilité de cocher « oui » ou
« non » à cette modalité sur le formulaire de consentement. Pour les
participants qui auront coché « oui », l'accord préalable du CNESVS de
la Côte d’Ivoire devra être obtenu pour toute nouvelle recherche portant
sur ces échantillons. L’utilisation de ces échantillons par des
chercheurs non mentionnés dans ce document devra être préalablement
approuvée par l’IPCI. Les participants qui auront coché « non » peuvent
tout de même participer à la recherche VAXPOX-CI, mais leurs
échantillons seront détruits à l’issue de la complétion des objectifs du
protocole.

Précisions sur les données nominatives collectées sur les sites
investigateurs

Pendant les enquêtes de séroprévalence, des informations identifiantes
seront collectées afin de constituer le registre des participants et de
permettre d’informer chaque participant de ses résultats individuels
(sérologie mpox). Ces informations identifiantes ne seront pas saisies
dans la base de données électronique du projet. Une table de
correspondance permettant de faire le lien entre l’identifiant unique
attribué aux échantillons prélevés et l’identité du participant sera
établie, conservée par les équipes terrain pendant la durée des
investigations, puis archivée sous enveloppe scellée à l’IPCI après les
investigations. Les résultats individuels du participant seront
communiqués par l’IPCI aux sites de recrutement de manière physique et
sécurisée. Les sites de recrutement seront alors responsables de la
restitution des résultats individuels. Pour le recrutement des
participants au sein des ménages de Dabou, effectué par les équipes
terrain de l’IPCI, les équipes se déplaceront pour retourner les
résultats individuels aux participants.

Précisions sur les données non-nominatives collectées par questionnaires
de recherche

La base de données du projet sera créée par la gestionnaire de données
de l’Unité d’Epidémiologie des Maladies Emergentes (UEME) de l’IPP via
l’outil REDCap, logiciel de recueil de données en ligne hébergé sur les
serveurs de l’IPP. REDCap est accessible à distance de façon sécurisée
par les utilisateurs référencés et authentifiés à l’aide d’un
identifiant et d’un mot de passe personnels, et permet de différencier
les droits d’accès (modification, lecture, extraction...) aux
formulaires et aux données qu’ils contiennent selon le profil des
utilisateurs (investigateurs, équipes d’investigations, gestionnaires de
données, attachés de recherche clinique). Un questionnaire au format
papier (CRF) reprenant les formulaires créés sur la base de données
électronique sera administré aux participants par les équipes de l’IPCI
ou des sites investigateurs. Les données seront saisies à posteriori
(double saisie) à l’IPCI dans la base de données électronique.

Précisions sur les données non-nominatives associées aux échantillons de
la biobanque du CNTSCI

Les données associées aux échantillons de la biobanque du CNTSCI, issues
des questionnaires administrés aux donneurs de sang lors des dons, sont
stockées sur les serveurs du CNTSCI. La sélection des données d’intérêt
pour le projet sera associée aux échantillons transférés à l’IPCI, et
communiquée électroniquement à l’IPCI par le CNTSCI de manière
sécurisée.

Conservation des données et archivage

L’IPCI, promoteur de cette recherche, conservera le registre des
participants, les consentements et CRF papier pendant 10 ans à compter
de la fin de l’étude (soit 12 ans au total). Passé ce délai, ces
documents seront archivés pour une durée de 5 ans avant destruction.

La base de données électronique du projet stockée sur les serveurs de
l’IPP sera conservée pendant 10 ans en base active à compter de la fin
de l’étude, puis archivée. Les autres données numériques reçues par
l’IPP seront également conservées pour une durée de 10 ans à compter de
la fin de l’étude (soit 12 ans au total), puis archivées.

Les données numériques enregistrées sur les serveurs de l’IPCI seront
conservées pour une durée de 10 ans à compter de la fin de l’étude (soit
12 ans au total), puis archivées.

Le rapport final de la recherche sera conservé par les deux instituts
pour une durée de 10 ans à compter de la fin de l’étude, puis archivé.

Lieux et analyses des données

Epidémiologie et sérologie (WP1-2)

L’analyse et l’interprétation des résultats de sérologie Luminex et des
données épidémiologiques associées seront effectuées conjointement par
les équipes des départements DPM et DESP de l’IPCI, avec l’appui des
unités Epidémiologie des maladies émergentes (UEME), Environnement et
risques infectieux (ERI), et Epidémiologie et analyse des maladies
infectieuses (IDEA) de l’IPP.

Au sein de chaque groupe de sujets ayant pris part aux enquêtes de
séroprévalence, les caractéristiques des sujets séropositifs et
séronégatifs pour les anticorps spécifiques d’un pathogène donné, dont
le MPXV, seront décrites par les moyennes et déviations standards, ou
les médianes et écarts interquartiles pour les variables continues, et
par les proportions avec intervalles de confiance à 95% pour les
variables discrètes. Le test du Chi-2 (ou test exact de Fisher selon les
effectifs observés) sera utilisé pour comparer les variables discrètes.
Le test de Student (ou test de Mann-Whitney selon les effectifs
observés) sera utilisé pour comparer les variables continues. Toutes les
variables associées en analyse univariée à un seuil ≤ 0,25 seront
intégrées dans une régression logistique pas à pas descendant afin de
déterminer les facteurs de risque de séroconversion. Ces facteurs de
risque seront mesurés par l’intermédiaire des Odds ratio (OR) et leur
intervalle de confiance à 95%. Le degré de signification statistique
retenu sera fixé à p<0,05.

Les données sérologiques obtenues et les données épidémiologiques
associées seront partagées dans une base de données commune avec les
autres instituts du PN déclinant le projet VAXPOX, permettant d’élargir
la surveillance collaborative entre les instituts du réseau à un volet
sérologique. 

Virologie moléculaire (WP3)

L’assemblage des séquences virales brutes afin de générer les génomes
entiers des souches MPXV issues des prélèvements positifs, ainsi que les
analyses phylodynamiques intégrant les métadonnées associées seront
réalisées par les équipes des départements DTT et DESP de l’IPCI, avec
l’appui des unités Epidémiologie et Physiopathologie des virus oncogènes
(EPVO) et Dynamiques évolutives des maladies infectieuses (EDID) de
l’IPP.

Les séquences et métadonnées associées seront déposées sur la plateforme
Owey (serveurs IPP) en suivant le format de pathoplexus
(https://pathoplexus.org/) dans lequel les séquences et les métadonnées
sont liées, et ainsi facilement déployées pour des analyses ou le
partage. Les données génomiques qui seront obtenues seront comparées
avec les autres génomes disponibles dans GenBank sur la plateforme Owey
(https://dataset.owey.io). Les analyses combinées des données de
phylogénie et des données épidémiologiques seront réalisées grâce à une
pipeline Nextstrain³⁶. Un modèle phylodynamique sera développé via la
plateforme BEAST2³⁷, utilisant toutes les séquences disponibles pour
décrire les dynamiques de transmissions zoonotique et interhumaine.

Modélisation mathématique (WP4)

Les données collectées et générées dans les autres WP seront utilisées
pour estimer des paramètres clés nécessaires au développement de modèles
mathématiques décrivant la circulation du mpox en Côte d’Ivoire. La
population modélisée sera structurée par âge et par groupe à risque
d’infection. Le modèle prendra en compte les différentes voies de
transmission, y compris la transmission interhumaine sexuelle ou non
sexuelle, ainsi que la transmission zoonotique. Un échantillonnage par
chaîne de Markov Monte Carlo sera utilisé pour calibrer le modèle aux
données disponibles. Grâce à cette calibration, la contribution des
différentes voies de transmission à la propagation globale sera
quantifiée. Le modèle obtenu sera utilisé pour simuler des épidémies
selon différentes stratégies de vaccination. Le fardeau d’infection
évité par chaque stratégie sera estimé, et celle qui maximise l’impact
de la vaccination sera identifiée. Le modèle sera calibré aux données
recueillies dans différents contextes (urbain vs rural) afin de
déterminer les caractéristiques spécifiques à chaque contexte, et
d’évaluer si les différences entre contextes justifient l’application de
stratégies de vaccination distinctes.

Des analyses coût-efficacité des stratégies de vaccination seront
réalisées, en tenant compte des différences de gravité de la maladie
selon les caractéristiques individuelles telles que l’âge ou le clade
viral. La charge d’infection évitée estimée par le modèle de
transmission sera convertie en mesures économiques ou sanitaires telles
que les ressources de santé utilisées ou les années de vie ajustées sur
la qualité. Cette harmonisation des résultats du modèle permettra de
calculer les ratios coût-efficacité incrémentaux et de fournir des
informations sur la valeur des stratégies de vaccination.

Ces travaux de modélisation seront effectués à l’IPP au sein des unités
Modélisation mathématique des maladies infectieuses (MMID) et
Epidémiologie des maladies émergentes (UEME), en collaboration avec les
modélisateurs du département DESP de l’IPCI.

Définitions

Événements indésirables

Un événement indésirable est défini comme toute manifestation nocive
chez un participant à une étude clinique, et qui n'est pas
nécessairement liée à la participation dans l’étude.

Événements / effets indésirables graves (EIG)

Un événement indésirable doit être considéré comme grave dès lors
qu’il : entraîne le décès, met en jeu le pronostic vital, entraîne une
incapacité ou une invalidité temporaire ou définitive, nécessite ou
prolonge une hospitalisation du patient, entraîne une anomalie
congénitale ou néonatale, est médicalement important (ce qui signifie :
nécessite une prise en charge pour éviter l’aggravation vers un des
stades ci-dessus).

Les événements suivants ne seront pas considérés comme EIG :

-   décès du fait de la progression d’une pathologie concomitante du
    participant (le cas échéant),

-   hospitalisations d’une durée inférieure à 24 heures (i.e. ne
    nécessitant pas une nuitée à l’hôpital),

-   hospitalisations qui auraient été planifiées avant la participation
    du sujet au protocole,

-   hospitalisations pour la prise en charge d’une pathologie
    concomitante diagnostiquée avant la participation du sujet au
    protocole, excepté si l’intensité ou la fréquence de celle-ci est
    augmentée depuis la participation du sujet au protocole,

-   hospitalisations pour des procédures cliniques de routine ou pour
    des raisons sociales.

Effets Indésirables attendus

Un effet indésirable attendu est un effet prévisible, lié aux actes
d’examens et de prélèvements requis par le protocole (pour les
prélèvements sanguins : douleur, hématome, malaise vagal).

Risques encourus par la recherche

La nature des actes médicaux réalisés est très peu à risque de
complication. Les seuls événements indésirables attendus sont liés à la
prise de sang : légère douleur au point de ponction au moment de
l’insertion de l’aiguille, petit hématome au point de prélèvement ou
malaise vagal bénin. Ces éventuels risques physiques seront minimisés
car les gestes seront pratiqués par du personnel formé à cet effet et
réalisés avec du matériel stérile à usage unique.

Recueil, notification et suivi des évènements indésirables 

Cette étude ne comportant que des risques et des contraintes minimes,
les éventuels évènements indésirables ne feront pas l’objet de
notification au promoteur mais seulement auprès du service concerné dans
le cadre du soin et dans lequel le participant pourra être adressé pour
une prise en charge adéquate.

Assurance

L’IPCI, en tant que promoteur, n’est pas tenu de souscrire un contrat
d’assurance spécifique pour cette recherche n’impliquant pas
l’administration de produits médicaux, conformément aux dispositions
légales et réglementaires ivoiriennes en matière de recherche sur la
personne humaine.

15. 1.  Monitoring

S’agissant d’une recherche impliquant la personne humaine à risques et
contraintes minimes prévoyant une collecte d’échantillons selon des
actes peu ou pas invasifs, aucun suivi étroit n’est jugé nécessaire.
Néanmoins pour satisfaire aux exigences des Bonnes Pratiques Cliniques,
les visites de monitoring suivantes seront réalisées sur site par un
attaché de recherche clinique en collaboration étroite avec le
promoteur.

Une réunion de pré-ouverture afin de :

- Vérifier les moyens techniques et les compétences des sites
investigateurs,

- Valider les modalités pratiques et logistiques.

- Définir, le cas échéant et en accord avec les centres, un plan
d’action en vue de la mise en place officielle de l’étude,

Une réunion d’ouverture afin de :

- Vérifier les moyens techniques et les compétences des sites
investigateurs,

- Rappeler les aspects scientifiques du protocole, la réglementation,

- Valider les modalités pratiques et logistiques.

Une visite de suivi par an, plus si nécessaire, afin de :

- Vérifier la validité des consentements,

- Vérifier ou collecter les données sur les fiches prévues à cet effet

- Vérifier la documentation des échantillons transférés.

Une visite de clôture administrative afin de :

- Vérifier la validité des consentements,

- Vérifier les fiches de recueil de données et la documentation des
échantillons transférés,

- S’assurer du contenu et de l’archivage du dossier investigateur.

- Pour réaliser ces visites, l’investigateur permettra un accès direct à
tout représentant du promoteur désigné à cet effet.

Correction du CRF

Les CRF ne peuvent être corrigés que par l’investigateur lui-même ou par
toute personne expressément désignée comme étant son délégataire. Chaque
correction doit être documentée par une date et les initiales de la
personne autorisée.

Audit

Dans le cadre des dispositions prises pour le contrôle et l’assurance de
la qualité de la recherche, le promoteur et les autorités de tutelle
pourront demander un accès direct aux données et aux documents sources,
et notamment au dossier médical des patients, pour vérification des
procédures et/ou des données des bases de données de surveillance de la
variole du singe réalisées par l’IPCI, sans violer la confidentialité et
dans les limites autorisées par les lois et réglementations.

L’investigateur ou un représentant dûment désigné doit être disponible
durant les visites de suivi, de monitoring ou d’audit organisé par le
promoteur.

la coordination clinique seront chargés de communiquer les ajustements
validés aux équipes de leur institut et de veiller à leur bonne mise en
œuvre.

Ce résultat montre qu’il n’y a pas dans le sang des anticorps qui
repèrent le virus de la maladie mpox (aussi appelée variole du
singe). Cela suggère que vous n’avez probablement pas eu de contact
passé avec ce virus, ni de vaccination, ou que les anticorps ne sont
plus détectables.

À l’avenir, si vous approchez une personne malade de mpox, appliquez
toutes les mesures de protection recommandées. Et si les autorités
sanitaires vous proposent une vaccination contre la mpox, nous vous
conseillons vivement de l’accepter.
